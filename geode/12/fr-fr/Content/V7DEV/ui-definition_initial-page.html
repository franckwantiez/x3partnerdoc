Initial page

You need a Sage ID to log in to Sage X3. However, there are two methods to log in to Sage X3, with two different initial pages. Use method 2 if you don't already have a Sage ID, or if your account is not registered.

Method 1: From http://www.sagex3.com/

You can log in to Sage X3 from http://www.sagex3.com/ by clicking the lock icon in the top bar.

You can then enter your Sage ID and password to log in.

This brings you to the gateway page which prompts you to select the instance you want to access.

The gateway also allows you to view messages regarding scheduled maintenance. For this reason, we recommend that customer administrators always use this method to log in to Sage X3.

After selecting your instance and clicking Continue, you are logged in to Sage X3 and your landing page is displayed.

Method 2: From a direct link to Sage X3

You can also log in to Sage X3 from the direct link you received in your welcome e-mail from Sage.

In this case, you get five options:

  • Oauth2: refer to this document for more information.
  • Do Not Use: not to be used.
  • Sign in with existing account: to be used if you already have a Sage ID and if your account is already registered.
  • Register existing account: to be used if you already have a Sage ID account but are logging in for the first time.
  • Create a new account: to be used if you do not have a Sage ID yet.

Select one of the options and follow the steps on-screen. You are logged in to Sage X3 and your landing page is displayed.
