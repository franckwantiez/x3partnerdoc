Tournées génériques

Cette table permet de définir l'ensemble des Tournées génériques. Cet écran permet de :

Consulter le détail d’une tournée générique.
Modifier le paramétrage d’une tournée générique.
Supprimer une tournée.
Créer une nouvelle tournée, c’est à dire prédéfinir des tournées de livraison « fixes ».

 

Pré-requis

Reportez-vous à la documentation de Mise en oeuvre

Gestion de l'écran

La saisie est réalisée sur un seul écran.

C’est une fonction facultative qui sera utilisée en fonction du paramétrage site.

Ecran de saisie

Pour créer une tournée générique, on attribue un code qui l’identifie de façon unique par site sur 10 caractères alphanumériques, et un nom.
Ces informations se trouvent en tête de l’écran de gestion. Figurent ensuite les informations liées à cette tournée générique :
Le code tournée générique est unique ; il peut exister n code tournées associées au même couple (transporteur/mode).

Code transporteur. Il s’agit du code identifiant le transporteur. Son existence est contrôlée dans la table des transporteurs.

Mode transport. Ce code est contrôlée dans la table des modes de transport. Généralement, le mode de transport correspond aux notions de Messagerie, Affrètement, Express, ..etc.  Il doit également être défini comme étant associé au code transporteur dans la table des transporteurs.

Type de port. Ce code est contrôlée dans la table des types de port.  Si renseigné, la tournée comprend soit des OL sans type de port, soit des OL avec le même type de port que celui indiqué dans la tournée.

Client livré (Disponible à compter de la G2). Ce code est contrôlée dans la table des clients. Si renseigné, la tournée comprend soit des OL sans code client livré, soit des OL avec le même code client que celui indiqué dans la tournée.

Maximum

Poids, volume, métrage linéaire maximum. Ces informations permettent de limiter le remplissage du camion, conteneur remorque, ... au delà de son gabarit. Un message fonctionnel avertira les opérateurs que les maxima sont atteints.

Délais de préparation. Permet de calculer lors de la recherche d’une tournée optimum pour l’O.L la tournée la plus adéquate.
Par exemple, il est 15 heures, la tournée passe à 17 heures, et pour cette tournée le délais de préparation est de 3 heures. Les OL seront donc affectés à la tournée suivante. Si les tournées génériques ne sont pas gérées, le délais de préparation des OL dans la table des critères transport est pris en compte.

Occupation des quais

Quai.(Disponible à compter de la G2). Ce code est contrôlée dans la table des magasins. Permet de préciser le quai de chargement de la tournée.

Durée. Durée d’occupation du quai pour la tournée générique. Lors de la visualisation du planning d’occupation des quais, le quai sera occupé à compter de l’heure de la tournée jusqu’à ce que al durée mentionnée soit atteinte.

Jours et heures limites

Permet de définir les jours et heures de la semaine de passage d’une tournée de livraison.

Inter site

inter site: permet de préciser si la tournée générique est de type "inter-site" ou non.

    • Si oui, la tournée générique est de type "inter-site" et elle a pour destination un autre site. Auquel cas, le champ suivant, "site de destination", doit être renseigné.
    • Si non, la tournée générique est à destination des clients finaux.

Site de destination: permet d'indiquer le site de destination d'une tournée générique réputée "inter-site". Ce champ est obligatoire lorsque le paramètre précédent est activé à 'oui'.

    • Le site de destination doit être différent du site courant.
    • Le site de destination doit appartenir au même dossier que le site courant.
Descriptions des champs

Bloc numéro 1

La tournée générique est une tournée de livraison « fixe » (par rapport aux jours, à l’heure limite d&rsq uo;expédition), une tournée étant le contenu d’un camion.

Code (champ GDR)
Intitulé (champ GDRDESAXX)
Intitulé court (champ GDRSHOAXX)
Actif (champ ENAFLG)

Sélectionnez cette case à cocher pour activer la fiche courante.

Les enregistrements non sélectionnés conservent leur contenu et paramétrage, mais ne pourront pas être utilisés en rappelant leur code dans :

  • les autres enregistrements, comme les documents et paramètres,
  • les traitements en masse.

Les habilitations sur une fonction donnée peuvent interdire la création d'une fiche active. Dans ce cas, la case est désactivée par défaut. Elle est modifiable uniquement par un utilisateur autorisé, ou via un Workflow de signature.

Délai de préparation (champ PRPLTI)

Lors de l'affectation automatique d'un OL à une tournée, le traitement prend en compte ce paramétre ajouté à l'heure de lancement du traitement pour déterminer la tournée la plus optimale.

Si les tournées génériques sont gérées, c'est le paramètre de la tournée qui est pris en compte, sinon c'est celui de la table des critères transport.

Si il n'y a pas d'affectation automatique des tournées, ce paramétre n'est pas utilisé.

Messages d'erreur

  • Messages génériques

Outre les messages génériques, les messages d'erreur suivants peuvent apparaître lors de la saisie :

Transporteurs, fiche inexistante

On a tenté de saisir un transporteur inexistant.

Modes transport, fiche inexistante

On a tenté de saisir un mode de transport inexistant.

Transporteurs, zone obligatoire

Le transporteur doit obligatoirement être renseigné.

Modes transport, zone obligatoire

Le mode de transport doit obligatoirement être renseigné.

Type de port, fiche inexistante

On a tenté de saisir un type de port inexistant.

Client livré, fiche inexistante

On a tenté de saisir un client inexistant.

Quai, fiche inexistante

On a tenté de saisir un quai inexistant.

Mode de transport non associé au transporteur XXX.

Le mode de transport saisis n’est pas associé au transporteur dans la table des transporteur.

Heure incorrecte

La valeur saisie est supérieure à 23 :59.

Erreur zone "Site de destination": Fiche inexistante

Le site de destination saisis n'appartient pas au même dossier que le site courant.

Le site de destination ne peut pas être le site courant.

Le site de destination de la tournée générique doit être différent du site courant.
