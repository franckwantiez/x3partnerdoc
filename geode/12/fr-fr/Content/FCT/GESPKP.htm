Ordres de préparation

Cet écran permet de visualiser le détail d’un O.P quel que soit son état.

L’O.P comporte un statut permettant la visualisation de l’état d’avancement.

En entrant dans la fonction, seuls les O.P à traiter (à l’état  1- non validé ou 3- partiellement validé) sont affichés. Pour afficher les O.P déjà traités (état 4-Validé), faire Sélection puis Tous.

Cet écran permet aussi de visualiser et d’imprimer la charge d’un OP actif (non validé).

Pré-requis

Reportez-vous à la documentation de Mise en oeuvre

Gestion de l'écran

  • En-tête
  • Onglet Ordre de préparation
  • Onglet Mouvements

L’écran est constitué de 3 onglets : un onglet d’identification de l’OP, un onglet avec les informations générales de l’OP et un onglet listant les mouvements de sortie de l’OP.

En-tête

Cet onglet présente les informations d’identification de l’ordre de préparation : le numéro O.P, le numéro de vacation qui a généré l’O.P, la date et heure de création de l’O.P et l’état.

Le Numéro d’OP est affecté automatiquement à la création de l’entité.

Le code état donne l’état d’avancement de l’OP.

Onglet Ordre de préparation

 L’entête de l’O.P comporte des informations générales caractérisant l’O.P et le mode de préparation :

L’O.P est de type « kit » s’il comprend des lignes de type composé et composant.
Les OP kit sont générés mixte, multi zones de préparation, multi niveaux et ceci quel que soit le paramétrage du mode prépa associé.

L’O.P est mixte, Picking ou réserve selon le paramétrage du mode de préparation (et non le prélèvement réel).

La zone de préparation est renseignée si le mode de préparation est mono zone de préparation, sinon la zone renseignée est celle du mode de préparation.

Le N° d’O.L est renseigné si le mode préparation est mono O.L.

Le N° de tournée est renseigné si le mode préparation est mono tournée.

Le Niveau est renseigné si le mode préparation est mono niveau.

 Il comporte aussi des informations de dimensionnement de l’OP :
- le nombre de colis de l’OP si gestion du précolisage ou le nombre de mouvements.
- le nombre d’UC de l’OP
- le poids et le volume de l’OP
- le temps de préparation de l’OP
- l’équivalent surface de l’OP si gestion du précolisage
