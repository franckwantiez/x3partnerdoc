Les sites de base

XTEND est livré en standard avec un site modèle (ATEMPLATE) et un site d'exemple (ASAMPLE).

ATEMPLATE
C'est un site qui ne contient que les paramètres obligatoires pour assurer le bon fonctionnement d'XTEND.
Aucune interface ni page HTML ne sont livrées avec ce site.

ASAMPLE
Le site ASAMPLE contient les tokens XTEND standards et un projet HTML avec des pages d'exemple (accueil, à propos, login, contact, erreur....).

 Pour vérifier que le dossier X3 est correctement configuré et que les applications XTEND fonctionneront correctement, nous conseillons de commencer par faire fonctionner le site ASAMPLE sur votre serveur.

ASHOP
Le site ASHOP est un site exemple livré uniquement sur le produit Sage X3 dans le dossier de démonstration. Ce site montre une boutique en ligne.

Mise en oeuvre du site ASAMPLE

1. Vérifiez que le serveur de web services répond bien

2. Vérifier la présence des fichiers du projet HTML

3. Activer les liens avec le serveur X3

4. Vérifier que le site ASAMPLE est bien publié

      • Cocher 'Publier le site' dans la fiche site

5. Valider le site ASAMPLE

Cocher 'rapport détaillé', 'vérification' et cocher 'tous les éléments'

6. Vérifier qu'il n'y a pas d'erreurs dans la trace

7. Visualiser le site ASAMPLE ( 2 solutions)

      • Solution 1 : Visualiser le site à l'aide de la fonction X3
      • Solution 2 : Avec le navigateur Web utilisez l'url suivante :
        http://hostname:port/xtend/page?SITE=ASAMPLE&amp;SOL=X3SOL&amp;FLDR=X3FOLDER &amp;PAG=AHOME&amp;LANG=FRA ,         apres avoir changé hostname:port, X3SOL et X3FOLDER par des valeurs valides

Création d'un nouveau site

 Nous vous conseillons de toujours créer un nouveau site par copie d'un site existant pour disposer de tous les 'tokens' standards nécessaires au bon fonctionnement d'XTEND.

Pour créer un nouveau site vous pouvez partir:

  • soit du site ASAMPLE si vous souhaitez mettre en oeuvre les mêmes fonctions dans votre site
  • soit du site ATEMPLATE qui contient le minimum requis pour développer un nouveau site
  • soit du site ASHOP qui est un exemple de boutique en ligne livré avec le produit Sage X3.

Paramétrage

Ouvrir une session X3 sur le dossier dans lequel vous souhaitez créer le nouveau site XTEND.

1. Sélectionner le site ASAMPLE, ATEMPLATE ou ASAMPLE comme site courant

2. Dupliquer le site ASAMPLE, ATEMPLATE ou ASHOP vers le site MYSITE

      • Cocher paramétrage
      • Cocher design html si vous partez du site ASAMPLE ou du site ASHOP
      • Cette fonction crée le nouveau site MYSITE

3. Sélectionner le site MYSITE comme site courant

4. Valider le site MYSITE

      • Cocher 'rapport détaillé', 'vérification' et 'tous les éléments'

5. Vérifier qu'il n'y a pas d'erreurs dans la trace

      • Si une erreur se produit valider toutes les interfaces manuellement

6. Vérifier que l'url ci-dessous affiche bien la page d'accueil

      • http://hostname:port/xtend/SOL=X3SOL&FLDR=X3FOLDER&SITE=MYSITE&LANG=FRA
      • Changer hostname:port, X3SOL et X3FOLDER par des valeurs valides

Création du projet HTML

Si vous êtes parti du site ASAMPLE et que vous avez coché 'Design HTML' lors de la copie, le projet HTML (pages, images, css, javascripts...) du site MYSITE a été créé. Il est localisé sur le serveur X3.

Bien qu'il soit possible de travailler directement sur les pages HTML distantes (serveur X3) en FTP avec DreamWeaver ou bien avec un éditeur de texte nous vous conseillons fortement de travailler sur une copie locale et d'effectuer les mises à jour serveur par transfert FTP

Création d'une copie locale

1. Créer un répertoire local pour vos projets HTML XTEND

2. Créer un sous-répertoire 'MYSITE\FRA'
    'MYSITE' est le code du nouveau site et 'FRA' le code langue

3. Télécharger (download) le contenu du projet HTML MYSITE du serveur X3 dans ce répertoire

N'oubliez pas de 'uploader' sur le serveur X3 les fichiers modifiés pour qu'il soient pris en compte par le navigateur.
