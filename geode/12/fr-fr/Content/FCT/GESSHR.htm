Repalettisation

Cette fonction permet de repalettiser des UM entre eux.

Les actions suivantes peuvent être réalisées :

  • Regrouper des UMs dans une nouvelle UM.
  • Eclater des colis d'un UM dans une nouvelle UM ou dans une UM existante.
  • Modifier les caractéristiques d'une UM.

On ne peut regrouper/éclater que des UMs non fermées, non chargées, non expédiées et non reportées. Cependant la fonction propose et affiche les UMs fermées car elle donne la possibilité de les ouvrir lors d'une demande de regroupement ou d'éclatement.

Remarque: Toutes les actions ne sont pas toujours réalisables. Elles dépendent de l'origine des UM (prépalettisé, palettisé auto ou palettisé), de leurs états et caractéristiques. Plus bas dans cette documentation, vous trouverez 2 tableaux récapitulant les réalisations possibles pour chacune des actions.

Pré-requis

Reportez-vous à la documentation de Mise en oeuvre

Gestion de l'écran

  • En-tête
  • Onglet Repalettisation

L’écran est constitué:

  • d'un entête permettant de filtrer les UMs à afficher selon des critères de sélection.
  • d'un onglet "Repalettisation" listant les UMs et leurs colis associés.

En-tête

Cet entête permet de saisir des critères de sélections sur les UMs:

  • leur origine: Prépalettisation (UM créé lors d'une préparation en mode prépalettisation), Palettisation auto (UM créé sur une ordre de palettisation automatique) ou Palettisation (UM créé manuellement par palettisation standard).
  • leur état
  • le N° OP (Ordre de Préparation)
  • le N° OPA (Ordre de Palettisation Automatique)
  • le N° vacation
  • le N° tournée
  • le N° d'UM

Onglet Repalettisation

Cet onglet affiche les UM répondant aux critères de sélection de l'entête. Il est constitué de 2 blocs:

  • Le 1° bloc (UM) liste les UMs non chargées, non expédiées, non reportées, ouvertes et fermées.
  • Le 2° bloc (Colis) liste l’ensemble des colis de l'UM sélectionnée dans le 1° bloc.

Etats

Par défaut, les états suivants sont associés à la fonction :

  ETIQUMCOLIS : Etiquettes UM & colis

Mais ceci peut être modifié par paramétrage.

Boutons spécifiques

  • Modifier
  • Regrouper
  • Eclater
  • Recherche

Modifier

Etape 1: Sélectionner l'UM à modifier et actionner le bouton .

Etape 2: Si l'UM est fermée, une demande de validation d'ouverture est requise de la part de l'opérateur pour continuer l'opération de modification.

Etape 3: Un écran de saisie des informations UM est alors affiché. Après avoir saisi les informations UM, le système demande à l’utilisateur s’il souhaite re éditer une étiquette UM.

Regrouper

Etape 1: Sélectionner dans la colonne Rgp les N° d'UM à regrouper, puis actionner le bouton .

Etape 2: Si des UMs sont fermées, une demande de validation d'ouverture est requise de la part de l'opérateur pour continuer l'opération de regroupement.

Etape 3: Les colis de ces UMs sont alors regroupés dans une nouvelle UM, pour laquelle le système demande de siasir le type d'UM ainsi que les informations dimensions et poids réel.

L'état de la nouvelle UM sera celui de l'UM la moins avancée; les dimensions/poids seront mises à jour.

Etape 4: Une nouvelle étiquette UM est automatiquement éditée pour la nouvelle UM créée si la configuration des sorties du déposant courant le stipule.

Eclater

Etape 1: Sélectionner l'UM à éclater dans le bloc UM, puis sélectionner les colis dans le bloc Colis.

Il est possible de transférer la totalité des colis dans un autre UM ou d’éclater partiellement un colis vers un autre UM.

Etape 2: Une fois les éléments sélectionnés, actionner le bouton .

Etape 3: Si des UMs sont fermées, une demande de validation d'ouverture est requise de la part de l'opérateur pour continuer l'opération d'éclatement.

Etape 4: Un écran de saisie des informations UM est alors affiché. Il est possible de transférer vers un UM existant, en saisissant le N° d'UM, ou en transférant vers une nouvelle UM.

Si l'entête OL est associée à un type d'UM particulier, et que le nouveau type saisi diffère, un message non bloquant averti l’utilisateur.

Dans les 2 cas, une étiquette UM est éditée si la configuration des sorties du déposant courant le stipule.

Recherche

Cette option permet de lancer une recherche des UMs selon les critères de sélection préalablement saisis.

Messages d'erreur

  • Messages génériques

Outre les messages génériques, les messages d'erreur suivants peuvent apparaître lors de la saisie :

Erreur Zone "Rgp": UM No X (...) Impossible de regrouper avec UM No Y (....)

Il est impossible de regrouper ces UM à cause de caractéristiques différentes citées entre parenthèse dans le message d'erreur. Ces caractéristiques sont : UM mono destinataire ou non, Etat de l'UM, Origine des UMs.

Erreur Zone "N° UM": Colis No X (...) Impossible d'éclater sur UM No Y (....)

Il est impossible d'éclater le colis dans cette UM à cause de caractéristiques différentes citées entre parenthèse dans le message d'erreur. Ces caractéristiques sont : UM mono destinataire ou non, Etat du colis et de l'UM, Origine des colis/UMs.
