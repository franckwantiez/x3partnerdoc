Message Web

 Avant de créer votre site , il préférable de lire la documentation :Avant de commencer.


Cette fonction permet de modifier les messages utilisés par le serveur XTEND.

Ces messages sont différents de ceux qui sont renvoyés par les actions X3 via l'appel du sous-programmes 'ADDMESSERR'.

Les messages web sont gérés par le moteur XTEND et sont renvoyés lorsque le moteur constate une erreur de traitement ou de droit d'accès.

Prise en main

Les messages XTEND sont traités comme les messages applicatifs X3 et sont intégrés dans le processus de traduction.

Paramètres des messages

Utilisez la syntaxe ci-dessous pour ajouter des paramètres dans les messages :

Valeur du paramètre N°1={1} - Valeur du paramètre N°2={2}

Messages du moteur XTEND

Tous les messages dont le code commence par 'A' sont des messages du moteur XTEND.

Vous pouvez modifier ces message en leur associant un autre chapitre dans lequel vous aurez saisi les nouveaux textes en conservant les paramètres.

Afficher les messages dans la page HTML

L'affichage des messages utilisateur est effectué via le token AXUSERMSG.

Exemple :

<span id="userMsg" class="userMsg" adx="aMsgUser"></span>

Afficher les messages en JavaScript

Pour afficher un message qui n'est pas un message du moteur XTEND, appelez la fonction JavaScript xtdAlert(aId,aP1,aP2,aP3,aP4) à partir de la page HTML.

La fonction xtdAlert demande le message au serveur XTEND via une requête AJAX.

Paramètres

  • aId est le code du message XTEND
  • aP1,aP2.. sont les paramètres (facultatifs)

Le cas peut se produire par exemple lors du contrôle d'un formulaire HTML.

<script>
function xtdAlert(aId,aP1,aP2,aP3,aP4);
// L'exemple ci dessous affiche
// 'Vous êtes connecté à l'application avec le code utilisateur DIS001'
xtdAlert("ASESSLOGIN2","DIS001");
</script>

Boutons spécifiques

  • Validation
  • Copie

Validation

Ce bouton permet de mettre à jour le dictionnaire XTEND au format XML.

Le serveur X3WEB prend en compte uniquement le dictionnaire sous ce format.

Il existe un fichier xml par dictionnaire.

Par exemple le fichier ACT.xml correspondant à l'ensemble des actions web pour le site Web courant.

Copie

Ce bouton permet de copier un message Web
