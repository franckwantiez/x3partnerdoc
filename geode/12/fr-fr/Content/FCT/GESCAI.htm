Allées de circulation

Cette table permet de définir une correspondance entre les allées physiques constituant un magasin et les allées de circulation (ou travées) utilisés dans le cadre des prélèvements, des réapprovisionnements et/ou des comptages inventaires.

La définition de cette table permet donc d’optimiser les déplacement des opérateurs lors de la réalisation physiques des opérations.

En fonction de ces 3 types de mouvements, les tris des mouvements au sein d ‘une entité logique peuvent être différents ; si elle n’est pas renseignée le tri est réalisé par magasin /adresse.

Pré-requis

Reportez-vous à la documentation de Mise en oeuvre

Gestion de l'écran

A partir d’un magasin sélectionné, la saisie des allées de circulation est réalisé dans un tableau pré renseigné.

Ecran de saisie

Le magasin peut être saisi ou sélectionné : les allées physique si elles existent sont affichées.

Le type de mouvement définit le cadre dans lequel sont utilisées les allées de circulation.

- Mouvement de type prélèvement: tri des ordres de prélèvement, des mouvements dans un colis.
- Mouvement de type réapprovisionnement: tri des listes de réapprovisionnement.
- Mouvement de type inventaire: tri des comptages d’inventaire.

Allées de circulation

Allée. L’allée est renseignée par rapport à la structure d’adresse du magasin et aux emplacements réellement créés.

Parité. Ce paramètre précise la parité associée à l'allée. Cette zone n'est accessible à l'opérateur que si les adresses du magasin sont de type « peigne ».

Impaire : Rangées impaires du code allée.
Paire : Rangées paires du code allée.

Une rangée est paire si le dernier caractère de la rangée est paire ; un caractère est pair si sa valeur ASCII est paire. Pour les chiffres de 0..9, les valeurs ASCII vont de 48..57. Pour les lettres de A..Z, les valeurs ASCII vont de 65..90. Les allées dont le dernier caractères est 0, 2, 4 ..,8 ou B, D, F et Z sont paires.

Circulation. Cette zone correspond à l'allée de circulation, elle permet le tri des mouvements à l’intérieur d’un OP. Deux types de circulations sont définies :

Circulation mono face : chaque allée doit avoir une allée de circulation unique (voir diagramme).
Circulation multi face : les allées « préparées » en m^me temps doivent avoir la même allée de circulation.
 
Niveau / Rangée. Cette zone définit le type de circulation à l’intérieur d’une allée de circulation. Tri par niveau puis par rangée.

Rangée / Niveau. Cette zone définit le type de circulation à l’intérieur d’une allée de circulation. Tri par rangée puis par niveau.

Sens rangée. Ce paramètre, sert à forcer le sens de parcours d'une allée soit dans le sens croissant, soit dans le sens décroissant ou alterné des rangées.

Sens niveau. Ce paramètre, sert à forcer le sens de parcours d'une allée soit dans le sens croissant, soit dans le sens décroissant ou alterné des niveaux.

Remarque : Si le sens est alterné, le système alterne sens montant et sens descendant une rangée sur deux
