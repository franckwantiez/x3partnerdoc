Ajustements suppr en masse

Utilisez cette fonction pour réaliser des ajustements de type suppression sur plusieurs objets de stock à la fois.

Pré-requis

Reportez-vous à la documentation de Mise en oeuvre

Gestion de l'écran

  • En-tête
  • Onglet Objets de stock

A l’ouverture de la fonction, une fenêtre de Critères de sélection s’affiche. Vous pouvez l’utiliser pour filtrer les objets de stock que vous voulez voir apparaître dans l’écran des ajustements de suppression en masse qui s’affiche par la suite.

Vous devez :

1. entrer le code mouvement,
2. sélectionner des objets de stock, avec saisie des numéros de série à sortir si nécessaire,
3. lancer le traitement d’ajustement de suppression, avec saisie possible d’un code opérateur selon le paramétrage du site.

Descriptions des champs
Code mvt (champ MVT)

Le code mouvement permet de différencier les mouvements de même type.
Il est requis lors de la création :

  • d'une entrée attendue,
  • d'une entrée fournisseur,
  • d’une entrée usine,
  • d’un retour client,
  • d'un O.L,
  • d'un ajustement,
  • d'un transfert,
  • d'une entrée ou sortie de consommable.

Le code mouvement est vérifié par rapport à la table des codes mouvements.
Date (champ DAT)

Date de création de la transaction.
Heure (champ TIM)

Heure de création de la transaction.

Onglet Objets de stock

Le tableau affiche les objets de stock unitaires du site.
L’icône Maximiser située dans la partie supérieure de la barre d’outils permet d’afficher le tableau sur l’ensemble de la fenêtre.
Seuls les objets de stock sans en cours (qui ne sont pas en cours de stockage…) et localisés dans des emplacements de type « réserve » sont affichés.
Vous pouvez sélectionner les objets de stock :

  • un par un en cochant les cases correspondantes,
  • en masse en cliquant sur l’icône Actions, située dans la deuxième colonne, puis Tout cocher.

Pour valider la création de l’ajustement de suppression, cliquez sur Créer dans la barre d’Actions

Emplacements

Pour les objets de stock dont l’article géré des numéros de série, vous devez indiquer les numéros de série à sortir pour effectuer l'ajustement de type suppression. Sur chaque objet de stock sélectionné, vous devez alors indiquer autant de numéros de série qu’il y a d’UC (Unité de consommation) sur cet objet à supprimer.
La procédure à suivre est la suivante : 


1. cliquez sur l’icône Actions de la ligne de l’objet de stock,
2. cliquez sur Saisie numéros de série : une fenêtre s’ouvre,
3. cliquez sur l’icône Actions à côté des numéros de série et cliquez Supprimer,
4. cliquez Enregistrer.

 Les champs Qté à modifier et Qté modifiée indiquent les numéros de série restants à modifier et se mettent à jour automatiquement lorsqu’un numéro de série est supprimé.

Barre de menu

Critères



Cliquez sur Critères pour accéder à l’écran de sélection des objets de stock. Trois types de critères peuvent être utilisés :
  • Les critères Emplacement.
  • Les critères Article.
  • Les critères Objets de stock.
Vous pouvez saisir plusieurs types de critères à la fois. 

Messages d'erreur

  • Messages génériques

Outre les messages génériques, les messages d'erreur suivants peuvent apparaître lors de la saisie :

Code mouvement interdit

Le code mouvement saisi n’est pas un code de type ajustement suppression.

Magasin : fiche inexistante

Le magasin saisi n’existe pas.

Format incorrect

Le format de l’adresse saisie ne correspond pas à celle du magasin.

Emplacement : fiche inexistante

L’emplacement saisi n’existe pas dans le magasin sélectionné.

Article : fiche inexistante

Le code article saisi n’existe pas.

Pas de fiche sélectionnée

Les critères saisis emplacement/article ne permettent pas de sélectionner un objet de stock.

Code mouvement : fiche inexistante

Le code mouvement saisi n’existe pas pour ce déposant.

N° de lot : zone obligatoire

L’article doit être géré avec un N° de lot, qui n’a pas été saisi.

Pas de saisie ou saisie incompléte des numéros de série

Des objets de stock sélectionnés pour suppression n'ont pas fait l'objet de la saisie complètes des numéros de série.

Un message d’erreur s’affiche lorsque vous tentez de supprimer ces objets de stock sans avoir au préalable entré le numéro de série.
