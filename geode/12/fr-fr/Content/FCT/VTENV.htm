Cette fonction affiche les sites/déposants accessibles pour l’utilisateur.
Le curseur se positionne sur le site/déposant courant.
La 3ème colonne accessible par    affiche l’intitulé du déposant.

Pour modifier le site/déposant courant

Déplacer le curseur jusqu’au site/déposant souhaité, avec les flèches de déplacement haut et bas, puis valider avec  . Les informations affichées dans les fonctions ultérieurement utilisées proviennent dès lors du site/déposant modifié.

Pour modifier le site/déposant par défaut

Une question propose que le site/déposant choisi devienne le site/déposant par défaut lors des prochaines connexions de l’utilisateur dans l’application.
