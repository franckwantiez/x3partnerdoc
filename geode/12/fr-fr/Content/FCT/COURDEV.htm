Cours des devises

Cette fonction permet de définir les cours de devises et de consulter l'historique de ces cours sous forme graphique et tableau. Un cours de devise est défini :

  • pour une devise donnée,
  • pour un type de cours donné,
  • à une date donnée.

On peut alors associer, à chaque devise de destination donnée, un cours (cours direct), le cours inverse étant affiché.

Pré-requis

Reportez-vous à la documentation de Mise en oeuvre

Gestion de l'écran

  • En-tête
  • Onglet A date
  • Onglet Historique

Dans l'écran, les 3 informations de départ (devise origine, type cours et date cours) sont saisies.

Toutes les devises pour lesquelles il existe un cours défini à cette date ou à une date antérieure sont alors affichés. Par ailleurs, ces cours peuvent être modifiés (la date d'application étant affichée).

Il est possible de :

  • modifier un cours pour une devise déjà saisie (la date de cours correspondant au cours modifié reste la même)
  • saisir une nouvelle ligne dans le tableau, y compris avec une devise déjà existante, afin de saisir un nouveau cours à une nouvelle date.

En-tête

Descriptions des champs
Devise origine (champ CURDEN)
Date cours (champ DATCOURS)
Type cours (champ CHGTYP)

Onglet A date

Les cours de devise sont saisis sous la forme de lignes identifiées par le cours de la devise de destination et la date.

Le système vérifie que, pour une devise donnée, il n'y a pas deux cours à la même date.

Descriptions des champs

Bloc numéro 2

Icône Actions
Initialisation devises

Permet de remplir le tableau avec une ligne par devise susceptible d'avoir un cours variable avec la devise d'en-tête. Les lignes correspondent au dernier cours connu à la date donnée en tête.

Si aucun cours n'est connu à cette date pour une devise donnée, un cours unitaire est proposé dans le tableau.

Onglet Historique

Cet onglet présente sous forme "tableau" et "graphique" les cours des devises sélectionnées. 
 Ce bouton permet de basculer du mode graphique au mode tableau.
En mode graphique, les dates sont représentées en abscisses, et les cours des devises, en ordonnées.
La roue permet à l'utilisateur de choisir le type de la courbe.
En outre, l'utilisateur peut choisir d'afficher toutes les courbes ou d'en masquer certaines (grâce à la barre dépliable dans le coin supérieur droit du tableau).

Boutons spécifiques

Bloc numéro 1

  • Devise origine (champ CURDEN)
  • Type cours (champ CHGTYP)
  • Nombre de jours d'historique (champ HISNBD)

Tableau Devises à afficher

  • Devise (champ CUR)

Ce bouton permet à l'utilisateur de visualiser les courbes des cours des devises sélectionnées, pour le nombre de jours d'historique demandé.
Après chaque changement de critères, le système rafraichit automatiquement le graphique avec la nouvelle entrée.
L'utilisateur peut mémoriser les critères renseignés pour un graphique donné via le bouton Mémo.
 La sélection des critères ainsi que le rappel des mémos n'ont aucun impact sur la liste des devises affichées dans l'onglet "A date".

Messages d'erreur

  • Messages génériques

Outre les messages génériques, les messages d'erreur suivants peuvent apparaître lors de la saisie :

Code déjà existant en ligne :

Deux lignes ont été saisies avec le même code devise et la même date.

Devise euro, cours fixe :

Une devise dont le cours est fixe par rapport à l'euro a été saisie.
