Processus

Un processus est un graphique qui:

  • explique l'enchainement de fonctions Sage X3,
  • intégre des liens de type tunnel vers ces fonctions.

Cette fonction permet de décrire ce processus à l'aide d'un éditeur visuel.

Vous devez ensuite lier ce processus soit à :

  • une vignette du portail,
  • un menu processus.

Onglet Description

Descriptions des champs

Bloc numéro 1

Type (champ TYPEP)

Type du processus :

  • Menu : Type de processus particulier. Il est référencé pour être affecté dans un navigateur de processus (menu processus). L'éditeur n'a aucune utilité dans ce cas.
  • Processus : Processus standard décrit à l'aide de l'éditeur.
Code d'accès (champ CODACC)

Saisissez si nécessaire un code d'accès associé au champ courant de l'écran.
Code activité (champ CODACT)

Description

champ TEXTE

Description du processus au format HTML.

Onglet Dessin

Editeur visuel de processus

Formes

Cliquez sur le bouton pour insérer des formes dans le processus.

Vous pouvez ajouter :

  • un rectangle,
  • une ellipse,
  • un cercle,
  • une ligne.

Vous pouvez redimensionner les formes après les avoir ajoutées au processus à l'aide des poignées sur leurs contours.

La barre d'outils suivante apparait lorsque vous sélectionnez une forme :

A partir de cette barre d'outils, vous pouvez :

Actions Illustrations

Sélectionner la couleur de remplissage de la forme.

Sélectionner :

  • la couleur des bordures de la forme,
  • le style des bordures:
    • sans bordure,
    • traits pleins,
    • petits pointillés,
    • grands pointillés.
  • l'épaisseur des bordures.

Faire pivoter une forme par paliers de 45 degrés.

Saisir du texte dans une forme en double-cliquant dessus. Appuyez sur la touche Entrée de votre clavier pour sauvegarder le texte saisi.

Mettre en page le texte d'une forme en modifiant :

  • sa taille :
    • S,
    • M,
    • L,
    • XL,
    • XXL.
  • son style :
    • gras,
    • italique.
  • son alignement :
    • à gauche,
    • centré,
    • à droite,
    • par le haut,
    • par le bas.
  • sa couleur.

Associer un hyperlien à une forme.

Effectuer les modifications suivantes accessibles via le bouton ou via un clic droit sur la forme :
  • couper,
  • copier,
  • coller,
  • supprimer,
  • mettre la forme au premier plan,
  • mettre la forme en arrière plan, ...

Icônes

Cliquez sur le bouton pour afficher la liste des icônes disponibles. Vous pouvez redimensionner les icônes après les avoir ajoutées au processus.

Descriptions des champs

Bloc numéro 1

Module (champ MODULE)

Sélectionnez un module pour le paramétrage.

Ce champ vous permet de renseigner si l'écran doit être créé dans la base de données du dossier. Il l'est si le module auquel l'écran est rattaché est actif pour le dossier.

Bloc numéro 2

champ APROC

Editeur visuel de processus.

Se reporter à la section Dessin de l'aide en ligne pour plus d'informations sur cet éditeur.

Conseils de conception et d'utilisation

  • Lors de la saisie des intitulés, conmencez au milieu de l'ancien texte et non au premier caractère afin de conserver la mise en forme.
  • Groupez les objets à déplacer ensemble afin de conserver la mise en page.
  • Une zone de texte, par exemple Suivez le guide, peut être réservée à une brève explication du processus et des fonctions qui lui sont liées.

Remarques

Il est possible de définir des bibliothèques complémentaires d'images sur le serveur d'application pour disposer d'images spécifiques telles que des logos, des photos, des schémas particuliers. Ceci est décrit dans cette documentation technique annexe.

Boutons spécifiques

Import processus

Cliquez sur ce bouton pour importer un processus.

Validation

Cliquez sur ce bouton pour valider les modifications apportées à un processus.

Copie

Cliquez sur ce bouton pour copier le processus depuis votre dossier actuel vers un autre dossier.
