Ajustements

Cette fonction permet de raliser des comparatifs de stocks physiques (compts) et informatiques, et de gnrer les ajustements correspondants.éééé

Cette fonction créé/supprime ou modifie des objets de stocks de manière semi automatisée ; elle doit donc être utilisé par des utilisateurs confirmés.

A un instant t, les écarts de stocks sont calculés entre une photo du stock suite à la clôture du comptage et le stock saisi dans le comptage.

La visualisation des écarts est réalisée à partir des lignes de comptages, puis à chaque sélection d’une ligne le système indique les objets de stocks informatiques dans le bloc Objet de stock, puis propose les ajustements à effectuer dans le bloc Ajustement.

En-tête

Saisie d’un N° d’inventaire ainsi qu’un N° de comptage. L’inventaire doit être en cours, et le comptage clôturé.

Attention, le traitement peut se faire à partir de n’importe quel comptage clôturé, il est nécessaire de s’assurer de celui qui doit servir aux régularisations.

Le type de l’inventaire « par article » ou « par adresse » est affiché pour information.

Barre de menu

  • Annuler
  • Commentaire

Annuler

Ce bouton désélectionne les enregistrements et les ajustements mémorisés par le système et flage la ligne comme non « ajusté » de manière à ne plus la proposer ultérieurement.

Attention : si le bouton Annuler est utiliser après la sélection de toutes les lignes, le système ne proposera plus aucune ligne en écart !

Commentaire

Permet de saisir un commentaire relatif aux ajustements raliser. Dans la mesure o les ajustements ne sont pas confirms le commentaire saisi est dtruit.àéùéé

Messages d'erreur

  • Messages génériques

Outre les messages génériques, les messages d'erreur suivants peuvent apparaître lors de la saisie :

Code mouvement d’ajustement automatique non défini

Le code mouvement doit être défini dans la table du déposant courant.

Inventaire sans comptage clôturé avec écart

Le N° d’inventaire saisi ne comporte pas de comptage permettant de réaliser des ajustements, soit parce qu’il n’existe pas de comptage clôturé, soit parce qu’il n’y a pas d’écart

Il existe d’autres comptages clôturés.

Warehousing avertit l’opérateur qu’il existe d’autres comptages clôturés sur lesquels les écarts de stocks peuvent se baser.
