Actions ( Objet )
Sommaire général - Les modèles - Objet

A,B,C, D,E, F, G, H, I,K, L, M, N, O, P, Q, R, S, T, U, V, W, X, Y, Z

A
AB_CREATION Lors de l'abandon d'une création sur erreur ( après Rollback )
AB_MODIF Lors de l'abandon d'une modification sur erreur ( après Rollback )
ABANDON En abandon de création, duplication ou modification
AFFMASK Au premier affichage de l'écran
ANNULE Durant l'annulation d'enregistrement
AP_ANNULE Après l'annulation d'enregistrement
AP_CHANGE Après le changement de code
AP_FILGAUCHE Durant l'affichage d'une la liste gauche différente de la liste principale et des derniers lus.
AP_IMPRIME Après l'impression du document Crystal Report
AP_LISTE Après l'impression de la liste Crystal Report
AP_VERF_ANU Avant l'annulation d'un enregistrement ( après le contrôle des liens dictionnaire )
APRES_CHOI Après le choix d'un bouton, menu ou liste gauche, ...
APRES_CRE A la fin de la création ( transaction terminée )
APRES_MOD A la fin de la création ( transaction terminée )
APRES_MODIF Après toutes modification de champ
AV_ANNULE En début de transaction d'annulation
AV_IMPRIME Avant l'impression du document Crystal Report
AV_LISTE Avant l'impression de la liste Crystal Report
AV_VERF_ANU Avant l'annulation d'un enregistrement ( avant le contrôle des liens dictionnaire )
AVANT_ACT Avant l'exécution d'un bouton pré-défini
AVANT_CHOI Après le choix d'un bouton, menu ou liste gauche, ...
AVANT_DEVERROU Avant le déverrouillage d'une fiche ( modification terminée )
AVANT_MOD Lors du passage en modification ( après modification d'un champ )
AVANT_MODFIC Durant la modification d'un enregistrement ( début de transaction )
AVANT_OUVRE Première action exécutée
AVANT_SUITE Avant l'exécution d'une action déclarée à la suite d'une première action ( sur  bouton ou menu ) dans le dictionnaire des actions
AVANT_VERROU Avant le verrouillage d'une fiche
AVANTBOUT Avant l'exécution d'un bouton ou menu déclaré dans la fenêtre
Retour

 

B
BOITE Avant l'ouverture de la fenêtre avec ses onglets et ses listes de gauches
Retour

 

C
CHANGE Durant la transaction de changement de code
CLE_GAUCHE Avant l'affichage d'une liste gauche, pour modifier les caractéristiques de la clé de tri.
CREATION Durant la transaction de création
Retour

 

D
DEB_PICK Dès la sélection d'un élément dans la liste de picking
DEBUT_CRE En début de transaction de création ( OBJet combiné )
DEBUT_MOD En début de transaction de modification ( OBJet combiné )
DEFTRANS Début de l'analyse des fenêtres de l'OBJet
DEGRISE Après avoir dégriser les clés ( après création / modification )
DEPICK Après avoir désélectionner un élément dans la liste de picking
DEVERROU Lorsqu'on a déverrouillé une fiche ( modification terminée )
DROIT Avant l'alimentation des masques par la table principale
Retour

 

E
EFFMASK A l'effacement de masque ( abandon de création, ... )
ENV Sur un changement d'environnement ( pour GEODE & LOAN )
EXEACT Après l'exécution d'un bouton pré-défini, avant le traitement superviseur associé
EXEBOUT Après l'exécution d'un bouton déclaré en fenêtre
Retour

 

F
FERME Dernière action exécutée
FILGAUCHE Pour filtrer les enregistrements d'une liste gauche sauf celle des derniers lus.
FILTRE Définition des critères à utiliser en filtre sur la table principale
FIN Suite au bouton "FIN", avant le déverrouillage de l'enregistrement
FIN_ACTION Après l'exécution d'un bouton, menu ou liste gauche, ...
FIN_MOD A la fin de la modification ( transaction terminée )
FIN_PICK Fin du processus de sélection d'un élément dans la liste de picking
Retour

 

G
GAUCHE En sélection d'un élément de liste gauche simple ou hiérarchisée
GAUCHE9 En sélection d'un élément de liste gauche des derniers lus
GRISE Lorsqu'on vient de griser la clé ( après passage en modification )
Retour

 

H
HINT Avant l'affichage d'une liste gauche, sauf celle des derniers lus, pour indiquer une clé de parcours.
Retour

 

I
ICONE Dès le double click sur un icône en bas à droite de l'écran
IMPRIME Avant l'impression du document Crystal Report (pour GEODE )
INICRE Lors de la transaction de création, juste avant l'écriture
INIMOD Lors de la transaction de modification, juste avant la réécriture
Retour

 

L
LIENS Après la lecture d'un enregistrement
LIENS0 Avant la lecture d'un groupe d'enregistrements ( OBJet tableau et combiné )
LIENS2 Après la lecture d'un groupe d'enregistrements ( OBJet tableau et combiné )
Retour

 

M
MEMO2 Mémo picking dans la fenêtre de sélection picking ( pour GEODE )
MODIF Dans la transaction de modification, juste après la réécriture de l'enregistrement
Retour

 

O
OUVRE Après l'ouverture des tables et des écrans
OUVRE_BOITE Avant l'ouverture de la fenêtre
Retour

 

P
PICKE Après avoir sélectionné un élément dans la liste de picking
PRE_GAUCHE Page précédente, lorsque l'on pagine sur liste gauche
Retour

 

R
RAZCRE Lorsqu'on va créer une nouvelle fiche
RAZDUP Lorsqu'on va dupliquer une  fiche
REMP_DERLU Avant d'afficher la liste des derniers lus.
Retour

 

S
SETBOUT Désactivation des boutons et activation des menus
SETTRANS Fin de l'analyse des fenêtres de l'OBJet
STATUT Après l'exécution d'un menu déclaré en fenêtre.
STYLE Après la lecture et l'affichage d'un enregistrement
SUI_GAUCHE Page suivante, lorsque l'on pagine sur liste gauche
Retour

 

T
TIROIR Avant l'affichage de toute liste de gauche
TITRE Avant l'ouverture de la fenêtre
Retour

 

V
VARIANTE Analyse des fenêtres de l'OBJet (pour chaque fenêtre )
VERF_ANU Avant la transaction de suppression d'un enregistrement
VERF_CHG Avant la transaction de changement de code d'un enregistrement
VERIF_CRE Avant la transaction de création d'un enregistrement
VERIF_MOD Avant la transaction de modification d'un enregistrement
VERROU Lorsqu'on a verrouillé une fiche en modification ou suppression
Retour
