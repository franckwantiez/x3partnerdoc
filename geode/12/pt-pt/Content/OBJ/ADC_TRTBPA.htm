LOAD_CNT

Este ponto de entrada permite memorizar os contactos no I ecrã CNTxxx de l separador contacto

               Xxx =  BPR para gestão dos Terceiros

               Xxx =  BPP para gestão dos Prospects

               Xxx =  BPC para gestão dos Clientes

               Xxx =  BPS para gestão dos Fornecedores

               Xxx =  BPT para gestão dos Transportadores

               Xxx =  FCY para gestão dos Estabelecimentos

Contexto e modo de funcionamento.

Não existe transação em curso.
Não existe ficheiro de rasto aberto.
Este ponto de entrada está situado na etiqueta CHARG_CNT do tratamento TRTBPA.
Está chamado no momento de acesso a um Terceiro / Prospect / Cliente / Fornecedor / Transportador / Estabelecimento.
Permite assin de completar a alimentação do ecrã CNTxxx [M ] com os dados específicos contidos em CONTACTCRM [M :AIN]
Ou CONTACT [CNT]
L índice a utilizar no quadro do ecrã é  :nolign-1

LECT_BPA : Alimentação dos campos de ecrã a partir de [F:BPA]

Este ponto de entrada permite alimentar os campos de ecrã a partir das zonas da classe [F:BPA]

Contexto e modo de funcionamento

[The following tables are on line] :

Não existe transação em curso.
Não existe ficheiro de rasto aberto.
Este ponto de entrada está situado na etiqueta CHARG_BPA do tratamento TRTBPA.
É chamado no momento dos endereços do terceiro.
Permite assim completar a alimentação de ecrã [M] com os dados específicos contidos em BPADDRESS [M:BPA]
O índice a utilizar no quadro do ecrã é nolign-1

LECT_BPA_INTERSITE : Alimentação dos campos de ecrã a partir de [F:BPA]

Este ponto de entrada permite alimentar os campos de ecrã a partir das zonas da classe [F:BPA]

Contexto e modo de funcionamento

[The following tables are on line] :

Não existe transação em curso.
Não existe ficheiro de rasto aberto.
Este ponto de entrada está situado na etiqueta CHARG_BPA_INTERSITE do tratamento TRTBPA.
É chamado no momento do carregamento dos endereços do estabelecimento em mais do endereço do terceiro.
Permite assim completar a alimentação de ecrã [M] com os dados específicos contidos em BPADDRESS [M:BPA]

ECRIT_BPA : Alimentação das zonas de [F:BPA] a partir do ecrã

Este ponto de entrada permite alimentar as zonas de [F:BPA] a partir de ecrã no momento do registo dos endereços.

Contexto e modo de funcionamento

[The following tables are on line] :

Não existe transação em curso.
Existe um ficheiro rasto aberto.
Este ponto de entrada está situado na etiqueta CREBPA do tratamento TRTBPA.
É chamado no momento do carregamento dos registos dos endereços do terceiro.
Permite assim de completar a alimentação de BPADDRESS [M:BPA] com os dados específicos contidos no ecrã [M].

O índice a utilizar no quadro de ecrã é nolign-1.
