Operadores

A gestão dos operadores permite identificar um conjunto de pessoas que trabalham sobre o estabelecimento e definir-lhes autorizações em relação a cada posto de trabalho.

Em função da parametrização, um código operador será obrigatório aquando da validação dos fluxos.

Gestão do ecrã

O ecrã é constituído de um único ecrã de registo.

[Tab] Ecrã de registo

Cada operador criado deve ser identificado de maneira única por um código dos 5 caracteres alfanuméricos e um título.
Estas duas informações encontram-se em cabeçalho do ecrã de gestão.

Reencontram-se seguidamente diversas opções que permitem conhecer os actos de gestão que um operador é susceptível de realizar.
Estas informações são as seguintes:

Autorizações

Recepção. Se esta check box foi assinalada, o operador é autorizado de validar uma recepção.

Arranjo. Se esta check box for assinalada, o operador é autorizado de validar uma lista de arranjo.

Ajustamento manual. Se esta check box foi assinalada, o operador é autorizado de validar um ajustamento.

Transferência. Se esta check box foi assinalada, o operador é autorizado de validar uma transferência.

 Aprovisionamento. Se esta check box está assinalada, o operador está autorizado a validar um aprovisionamento.

 OP. Se esta check box está assinalada, o operador está autorizado a validar uma ordem de cobrança.

 Embalagem. Se esta check box está assinalada, o operador está autorizado de validar uma embalagem (pré-embalamento).

 Empacotamento declarativo. Se esta check box está assinalada, o operador está autorizado a criar e modificar pacotes de maneira declarativa (empacotamento declarativo).

 EDI transportador. Se esta check box está assinalada, o operador está autorizado a registar manualmente mensagens EDI.
