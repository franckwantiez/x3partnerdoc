Os estabelecimentos de base

XTEND está entregue em standard com um estabelecimento modelo (ATEMPLATE) e um estabelecimento de exemplo (ASAMPLE).

ATEMPLATE
É um estabelecimento que não contém que os parâmetros obrigatórios para assegurar o bom funcionamento de XTEND.
Nenhuma interface nem página HTML não estão entregues com este estabelecimento.

ASAMPLE
O estabelecimento ASAMPLE contém os tokens XTEND standards e um projeto HTML com as páginas de exemplo (acolho, a propósito, login, contacto, erro....).

 Para verificar que o dossier X3 está corretamente configurado e que as aplicações XTEND funcionam corretamente, nós aconselhamos de começar para fazer funcionar o estabelecimento ASAMPLE sobro vosso servidor.

ASHOP
O estabelecimentoASHOP é um estabelecimento exemplo entregue unicamente sobre o produto Sage X3 no dossier de demonstração Este estabelecimento mostra uma loja em linha.

Colocação em obra do estabelecimento ASAMPLE

1. Verifique que o servidor de web services responde bem

2. Verificar a presença dos ficheiros do projeto HTML

3. Ativar as ligações com o servidor X3

4. Verificar que o estabelecimento ASAMPLE está bem publicado.

      • Assinalar "Publicar o estabelecimento" na sua ficha estabelecimento

5. Validar o estabelecimento ASAMPLE

Assinalar "reporte detalhado", "verificação" e assinalar "todos os elementos"

6. Verificar que não existem erros no rasto

7. Visualizar o estabelecimento ASAMPLE (2 soluções)

      • Solução : Visualizar o estabelecimento de ajuda da função X3
      • Solução 2: Com o navegador Web utilise a url seguinte :
        http://hostname:port/xtend/page?SITE=ASAMPLE&amp;SOL=X3SOL&amp;FLDR=X3FOLDER &amp;PAG=AHOME&amp;LANG=FRA ,         após ter alterado hostname:port, X3SOL e X3FOLDER por valores válidos.

Criação de um novo estabelecimento

 Nós vos aconselhamos de sempre criar um novo estabelecimento por cópia de um estabelecimento existente para dispor de todos os "tokens" standards necessários ao bom funcionamento de XTEND.

Para criar um novo estabelecimento você pode partir :

  • seja do estabelecimento ASAMPLE se desejou colocar em obra as mesmas funções no seu estabelecimento.
  • seja do estabelecimento ATEMPLATE que contém o mínimo requerido para desenvolver um novo estabelecimento
  • seja do estabelecimento ASHOP que é um exemplo de loja em linha com o produto Sage X3.

Parametrização

Abrir uma sessão X3 sobre o dossier no qual se deseja criar o novo estabelecimeto XTEND.

1. SElecionar o estabelecimento ASAMPLE, ATEMPLATE ou ASAMPLE como estabelecimento corrente

2. Duplicar o estabelecimento ASAMPLE, ATEMPLATE ou ASHOP versus o estabelecimento MYSITE

      • Assinalar parametrização
      • Assinalar design html se parte do estabelecimento ASAMPLE ou do estabelecimento ASHOP
      • Esta função cria o novo estabelecimento MYSITE

3. Selecionar o estabelecimento MYSITE como estabelecimento corrente

4. Validar o estabelecimento MYSITE

      • Assinalar "reporte detalhado", "verificação" e assinalar "todos os elementos"

5. Verificar que não existem erros no rasto

      • Se um erro se produz validar todas as interfaces manualmente.

6. Verificar que URL a seguir afixa bem a página de acolho

      • http://hostname:port/xtend/SOL=X3SOL&FLDR=X3FOLDER&SITE=MYSITE&LANG=FRA
      • Alterar hostname:port, X3SOL e X3FOLDER por valores válidos

Criaação do projeto HTML

Se partiu do estabelecimento ASAMPLE e que assinalou "Design HTML" no momento da cópia, o projeto HTML (páginas, imagens, css, javascripts...) do estabelecimento MYSITE foi criado É localizado sobre o servidor X3

Ben que seja possível de trabalhar diretamente sobre as páginas HTML distantes /servidor X3) em FTP com DreamWeaver ou bem com um editor de texto nós aconselhamos fortemente de trabalhar sobre uma cópia loval e de efetuar as atualizações servidor por transferência FTP.

Criação de uma cópia local

1. Criar um diretório local para os projectos HTML XTEND

2. Criar um sub-diretório "MYSITE\FRA"
"MYSITE" é o código do novo estabeleicmento e "FRA" o código da língua

3. Telecarregar (download) o conteúdo do projeto HTML MYSITE do servidor X3 neste diretório

Não esqueça de "uploader" sobre o servidor X3 os ficheiros modificados para que sejam tomados em conta pelo navegador.
