Utilitários X3

Estabelecimentos Web

Função

Carregar sobre a tecla F7 e o nome da função GESAYS : Estabelecimento Web

Descrição

É uma função de parametrização do estabelecimento XTEND

Eis alguns eleementos parametrizáveis :

    • Tornar acessível o estabeelcimento (publicável),
    • o artualizar em manutenção ou não.
    • definir a lista das línguas,
    • tornar ativo os utilitarios de seguimento, debugging,...
    • ...

Selação do estabelecimento de trabalho

Função

Carregar sobre a tecla F7 e o nome da função AYTCHGFCYW :  Selecção estabelecimento corrente

Descrição

Para começar a parametrização XTEND é obrigatório de selecionar o estabelecimento de trabalho ou estabelecimento corrente.

Todas as funções de parametrização fazem a referência ao estabelecimeto de trabalho.

A função afixa a lista dos estabelecimentos XTEND existentes.

A seleção corrente está salvaguardada nas preferências utilizador X3

Visualização do estaebelecimento Web

Função

Carregar sobre a tecla F7 e o nome da função VISUAYS : Visualização estabelecimento Web

Descrição

Falta que o estabelecimento já seleccionado.

Pode-se visualizar o estabelecimento com a função GESAYG : Páginas Web. Se posicionar em seguida sobre a páhina AHOMA e escolher o menu do alto : Utilitários / Visualização página web.

A página AHOME se executa numa janela X3. Se você clicou sobre o ícone ao alto, a página se visualiza no navegador web por defeito.

Caso de emprego

Função

Carregar sobre a tecla F7 e o nome da função CONSAYU : Caso de emprego

Descrição

Afixa a lista das fichas de parametrização XTEND selecinados pelo seu tipo

Para cada ficha afixa a lista das fichas de parametrizaação que fazem referência a esta ficha

Exemplo:

Um campo pode ser referenciado por uma entidade ela mesmo pode ser referenciada por um bloco que pode ser referênciado por uma página.

Cada nível (3 máximo) utiliza 3 colunas para afixar o código, o tipo (Niv.#-Tipo) e a descrição da ficha.

Utilização

Esta função é muito útil para:

  • Conhecer as dependências entre parâmetros XTEND
  • Efetuar as cópias agrupados de parâmetros versus um outro estabelecimento
  • Efetuar as supressões agrupadas

Validação estabelecimeto Web

Função

Pressionar a tecla F7 e o nome da função AYTFCYGEN : Validação estabelecimeto Web

Descrição

Valida os elementos selecionados de um estabelecimeto XTEND : atualização do dicionário XML e afixação de uma relação.

Verificação

Se uma interface está assoicado à ficha, verifica a coerência dos dados com os parâmetros do web services

Coselhos

Em fase de desenvolvimento é aconselhados de validar regularmente todos os elementos do estabelecimento.

Os erros de validação dos ele,mentos são afixados em vermelho na relação.

A validação gera um novo dicionário XML dos parâmetros XTEND.

O novo dicionário só está tomado em conta automaticamente sobre reload/F5 da página HTML no navegador que se a opção da ficha de parametrização do estabelecimento, "Técnica\Verificar as atualizações\Dicionário web está assinalada.

Se a opção não está ativada falta forçar o recarregamento do dicionário com url :
'http://hostname:port/xtend/svc/SolutionX3/DossierX3/SiteXtend/admin/reposit/reload'

Duplicação do estaebelecimento Web

Função

Carregar sobre a tecla F7 e o nome da função AYTFCYCOP : Duplicação do estaebelecimento Web
Descrição

Permite copiar (com substituição) todos os parâmetros e/ou o design Htmel de um estabelecimento XTEND (origem) versus um estabelecimento no dossier corrente (defeito) ou um outro dossier.

Exemplo:

Cópia de um estabelecimento de desenvolvimento versus o ambiente de produção.

Restrições

Esta função não copia os web services utilizadas pelo estabelecimento.

Ver a função Copiar dos web services.

Cópia web service

Função

Carregar sobre a tecla F7 e o nome da função AYTWSRGES :  Cópia web service

Descrição

Permite selecionar uma ou várias interfaces de um estabelecimento XTEND do dossier X3 corrente e de copiar as web services associadas versus um outro dossier

Para ativar a cópia falta seleccionar ao menos uma interface na lista.

Cópia de um sub-programa
  • Cópia de um web service tratamento L4G
  • Criaação da ficha "sub-programa"
  • Publicação do web service
Cópia de um web service objeto X3
  • Publicação do web service

Ajuda à criação dos campos

Função

Esta função está acessível via o menu utilitários das fichas de parametrização.

Descrição

Ela permite criar os tokens campos a partir :

  • da lista dos campos de uma tabela X3
  • da lista dos campos de uma vista X3
  • os parâmetros do web service associados a uma interface
Ecrã

Coluna

Descrição

Parâmetro

Código do campo Ide a table ou parâmetro do web service

Título

Título campo

Token campo

Código XTEND para parâmetro (mesmo código por defeito)
Registável se deseja criar um campo com um nome diferente do parãmetro (mapping)

Existe

Assinalado se o campo existe já no dicionário

Tipo

Tipo de campo (XTEND)
Registável na criação

Menu

Identificador do menu para o tipo MenuLocal
Registável se deseja criar um campo com um nome diferente daquele do parâmetro.

Dimensão

Número máximo de valores do parâmetro ou campo

Geração

Para os campos de dimensão superior a 1, indica se falta gerar um índice (NOMCHAMP + INDICE).
- Nenhum: cria um campo sem índice
- Todos: criado tantos campos indiciados que valores possíveis para este campo
- Um: cria um único campo com o índice indicado na coluna índice
Registável à criação

Índice

Índice do campo para cálculo do nome
Registável à criação se Geração=Um

Falta validar os campos após geração para que sejam tomados em conta pelo servidor XTEND.

A validação gera um novo dicionário XML dos parâmetros XTEND.

O novo dicionário não está tomado em conta automaticamente sobre reload/F5 da página HTML no navegador que se a opção da ficha de parametrização do estabelecimento "Técnica\Verificar a atualização\Dicionário web" está assinalada.

Senão falta forçar o recarregamento do dicionário com URL :
http://hostname:port/xtend/svc/SolutionX3/DossierX3/SiteXtend/admin/reposit/reload

Ajuda à criação das entidades

Função

Menu "Utilitários/Ajuda ao preenchomento" da ficha "Entidade"

Descrição

Esta função permite criar as entidades a partir dos parâmetros de uma interfaces.

1. Abrir a função Entidade

2. Criar uma nova Entidade

3. Selecione o tipo de interface

      • Para o tipo "acesso tabela" seleccionar o interface que será anexado à entidade

4. Activar o menu "Utilitários/Ajuda ao preenchimento"

Após ter selecionado o interface, a lista dos parâmetros reagrupados por grupos de publicação está afixado.

Ecrã

Coluna

Descrição

Parametrização de interface

Código do parâmetro do web service

Grupo

Gripo de publicação (sub-programa) ou código do bloco (objeto)

Título

Intitulado campo para os objetos X3

Token campo

Código do token campo XTEND
Registável à criação

Título

Intitulado campo XTEND

Geralmente uma entidade XTEND está associada a um grupo de publicaação (coluna grupo) (que representa um bloco ecrã para um objeto X3).

A operação de criação consiste a selecionar os campos por duplo-clique sobre a primeira coluna e a selecionar um token campo na coluna "Token campo".

Recomendações

Nós aconselhamos de criar todos os campos XTEND previamente via a função utilitário "Criação dos campos".

Se o campo não existe não se pode aceder à função "Token campo" por túnel para o criar.

Supressão estabelecimento Web

Função

Carregar sobre a tecla F7 e o nome da função AYTFCYDEL :  Supressão estabelecimento Web
Ação

Permiite suprimir os design HTML e/ou parametrização de um estaebelecimeto XTEND.

Restrições

Os web services não são suprimidos porque podem ser potencialmente utilizados por outros estabelecimentos

Devem estar suprimidos manualmente via a função GESAWE : Web service

Gestão design/Arquivo

Função

Carregar sobre a tecla F7 e o nome da função AYTFCYTAR :  Gestão design/Arquivo
Descrição

Esta função permite de Criar/Restaurar/Suprimir um arquivo (ficheiro .tar) que contém o design HTML de um estabelecimento XTEND.

Esta função é muito útil para copiar/restaurar o design HTML de um estabelecimento XTEND versus um outro acondicionamento ou de outro dossier X3.

Parametrização avaçada

Função

Carregar sobre a tecla F7 e o nome da função GESAYU : Parametrização avaçada

Descrição

Esta função apresenta a lista dos parâmetros que não estão acessíveis via as fichas de parametrização.

Ela está utilizada para a aplicação dos patches do servidor XTEND.

Geração listas de valores

Função

Carregar sobre a tecla F7 e o nome da função PRHBATCH : Geração listas de valores

Descrição

Esta função gera as listas de valores das divisas e dos países baseado sobre as tabelas TABCUR e TABLAN.

Ela serve de ecemplo para a geração das listas de valores por batch.
