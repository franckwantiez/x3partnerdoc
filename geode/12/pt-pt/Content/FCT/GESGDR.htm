Rondas genéricas

Esta tabela permite definir o conjunto Voltas  genéricas. Este ecrã permite:

Consultar o detalhe de uma volta genérica.
Alterar parametrização de uma volta genérica.
Suprimir uma volta.
Criar uma nova volta, ou seja predefinir voltas de entrega "fixas".

Gestão do ecrã

O registo é realizado sobre só um ecrã.

É uma função facultativa que será utilizada em função parametrização estabelecimento.

[Tab] Ecrã de registo

Para criar uma volta genérica, atribui-se um código que o identifica de maneira única por estabelecimento sobre 10 caracteres alfanuméricos, e um nome.
Estas informações encontram-se em cabeçalho do ecrã de gestão. Figuram seguidamente as informações ligadas a esta volta genérica:
O código girado genérico é único; pode existir n código girados associado ao mesmo conjunto (transportador/modo).

Código transportador. Trata-se do código que identifica o transportador. A sua existência é controlada na tabela dos transportadores.

Modo de transporte. Este código é controlado na tabela dos modos de transporte. Geralmente, o modo de transporte corresponde às noções de Serviço de mensagens, frete, de Expresso..etc. Deve igualmente ser definido como associado no código transportador na tabela dos transportadores.

Tipo de porte. Este código é controlado na tabela dos tipos de porte. Se registada, a volta compreende quer as OE(ordens de Entrega) sem tipo de porte, quer dos OE com o mesmo tipo de porte que a indicada na volta.

Cliente expedido (Disponível a contar da G2). Este código é controlado na tabela dos clientes. Se informada, a volta compreende quer dos OE sem código cliente entregue, quer as OE com o mesmo código cliente que a indicada na volta.

Máximo

Peso, volume, comprimento linear máximo. Estas informações permitem limitar preenchimento do camião, contentor reboque... para além do seu gabarito. Uma mensagem funcional avisará os operadores que os máximos são atingidos.

Prazos de preparação. Permite de calcular no momento da pesquisa de uma volta óptima para a O.E. volta mais adequada.
Por exemplo, são 15, a volta passa às 17 horas, e para este volta os prazos de preparação é de 3 horas. As OE por conseguinte serão afectadas à volta seguinte. Se as voltas genéricas não forem geridas, os prazos de preparação das OE na tabela dos critérios transporte é tidos em conta.

Ocupação dos terminais

Terminal. (Disponível a contar G2). Este código é controlado na tabela dos armazéns. Permite de precisar o cais de carregamento da volta.

Duração. Duração de ocupação do cais para a volta genérica. No momento da visualização do planeamento de ocupação dos cais, o cais será ocupado a contar da hora da volta até a duração mencionada seja atingida.

  • Activo ([field] ENAFLG)

Esta check box a assinalar permite ativar ou de desativar a ficha corrente sem portanto perder o seu conteúdo.

Uma ficha desativada não pode ser utilizada (por chamada do seu código) em outras fichas (documentos, parametrizações...), ou no tratamento de massa.

As habilitações sobre uma função dada podem interdir a criação de uma ficha ativa. Neste caso, a check box está desativada por defeito, e está modificável unicamente por um utilizador autorizado, ou via um circuito de assinatura definido por Workflow.
