Dicionário

Esta função permite copiar em massa um conjunto de elementos do dicionário de um dossier versus um outro. Estes elementos do dicionário podem ser definidos por check boxes a assinalar, por limites de nome, e por uma data (elementos atualizados mais recentemente que uma dada data).

É possível de filtrar unicamente os elementos a copiar afetados pela sua pertença a um módulo, por um código de atividade dado.

Enfim, a transferência de um elemento pode-se acompanhar da cópia dos tratamentos associados, e fase da validação do dicionário sobre os elementos copiados no dossier de chegada pode ser igualmente ser lançada.

Gestão do ecrã

[Tab] Ecrã de registo

Define-se então o dossier de partida, depois o dossier de chegada. Por defeito, o dossier corrente está proposto, mas é possível de dar o nome de um outro dossier acessível, com as sintaxes detalhadas numa documentação anexa.

Os elementos correspondentes aos dados a copiar estão em seguida registados sob a forma :

  • De uma série de check boxes a assinalar para significar que este tipo de elemento deve ser copiado (tabelas, vistas, ecrãs, tipos de dados, ações, funções, objetos, janelas, consultas, parametrização das tabelas diversas, mensagens, tratamentos, e listas podendo ser copiadas).
  • Limites de início e de fim para cada elemento, assim que uma data de início.
  • De um quadro dos módulos afetados, e de um código atividade eventual para filtrar unicamete os dados afetados.

Define-se enfim se deseja validar os elementos sobre o dossier de chegada após cópia.

O lançamento desta operação, que pode ser longa, se termina pela afixação de um ficheiro rastro.

Dossiers

  • Dossier origem ([field] DOSORG)

﻿Este campo permite de definir o dossier a partir do qual a ficha vai ser copiada. As sintaxes possíveis são descritas no anexo dedicado
  • Dossier destino ([field] DOSDES)

﻿Este campo permite de definir o dossier no qual a ficha vai ser copiada. As sintaxes possíveis são descritas no anexo dedicado

[Grid] Objecto a tratar

  • Tipo ([field] OBCDES)
  • A tratar ([field] OBCSEL)
  • De ([field] OBCSTR)
  • Até ([field] OBCEND)

[Grid] Módulos

  • Nr. ([field] NUMLIG)

Linha corrente do quadro
  • Módulo ([field] MODULE)

Este quadro dá a lista dos módulos para os quais se pode seleccionar os elementos a copiar.
  • Cópia ([field] COPMOD)

Se o campo é igual a Sim, os elementos do módulo selecionados por entanto são copiados.

Opções

  • Scripts associados ([field] COPTRT)

Permite de indicar que os tratamentos associados aos elementos copiados (ecrãs, acções, objectos, funções, consultas) são igualmente copiados.

Este indicador provoca também a cópia dos estados Crystal Reports associados aos elementos do dicionário dos estados quando este é copiado.
  • Validação do dicionário ([field] VALDIC)

Permite desencadear a validação dos elementos (tabelas, vistas, ecrãs, janelas, objectos, consultas, funções) no dossier de destino após a sua cópia. Esta validação não é possível que se o servidor corrente associado ao dossier é o mesmo que o servidor do dossier de destino. Se não é o caso, uma mensagem de erro assinalará este facto no rasto da cópia.

Filtros

  • Depois de ([field] DATDEB)

Se esta data está preenchida, apenas os dados cuja data de última modificação ou da data de criação é superior ou igual são tomados em conta no tratamento.
  • Actividade ([field] CODACT)

Se um código actividade está registável aqui, os únicos elementos marcados pelo código actividade em questão são copiados.

Sim a todo o resto

Quando tem um quadro de registo de escolhas múltiplas com uma escolha de resposta Sim / Não sobre cada linha, esta função permite de responder Sim a todas as linhas a partir da linha corrente.

Não resta nada

Quando tem um quadro de registo de escolhas múltiplas com uma escolha de resposta Sim / Não sobre cada linha, esta função permite de responder Não a todas as linhas a partir da linha corrente.

No ficheiro rasto

Esta função cria um ficheiro rasto na execução, ficheiro no qual se pode reencontrar os erros seguintes :

Validação impossível sobre máquina distante

A check box Validação foi assinalada, mas o servidor sobre o qual se encontra o dossier de destino não corresponde ao servidor sobre o qual se encontra o dossier de onde a cópia está lançada.

Abreviação já existente ...

A tabela que se deseja copiar a mesma abreviação que uma outra tabela do dossier de destino.

Existe já uma vista com o mesmo nome

A tabela que se deseja copiar a mesma abreviação que uma outra tabela do dossier de destino.

Existe já uma tabela com o mesmo nome

A tabela que se deseja copiar a mesma abreviação que uma outra tabela do dossier de destino.
