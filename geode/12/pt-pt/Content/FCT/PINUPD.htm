Cálculo mnt. Transporte

No caso de modificação da tarifação de um transportador, pode ser interessante de re calcular os custos de transportes associados às entidades expedidas.

Esta função permite de re - calcular os novos custos e de actualizar as entidades BT/RT/BL no seguimento do novo cálculo.

Pré-requisitos

[Refer to documentation [Implementation]]

Gestão do ecrã

  • Cabeçalho
  • [Tab] BT
  • [Tab] RT
  • [Tab] GR

Registo dos critérios de selecção das entidades expedidas, seguidamente Botão  para apresentar os resultados da selecção, seguidamente botão  para apresentar o custo calculado.

O utilizador pode seguidamente validar este cálculo pelo botão , neste caso os custos calculados são actualizados nas entidades apresentadas.

Cabeçalho

Selecção

O período de expedição é obrigatório.
O transportador e o modo de transporte são facultativos mas permitem seleccionar entidades do mesmo modo transportador, modo.

Data

A data de cálculo permite indicar se o algoritmo de cálculo deve basear-se sobre a data de expedição da entidade e ter em conta assim a tarifa à data de expedição de cada entidade, ou se a data de cálculo deve se basear em relação a uma outra data a indicar (por exemplo a data do dia).

Seleção

Data de início do período de expedição. Por defeito a data é a data do dia. Esta é passível de alteração.

Data de fim do período de expedição. Por defeito a data é a data do dia. Es ta é passível de alteração.

  • Transportador ([field] CRR)

Código identificando o transportador. Este código deve ser definido na tabela dos transportadores.

  • Modo transpor. ([field] CRRMOD)

Geralmente, o modo de transporte corresponde às noções de servidor de mensagem, fretagem, expresso,... O modo de transporte está verificado por relação à tabela dos modos de transporte.
  • Data expedição inicio ([field] SHPSTR)
  • data expedição fim ([field] SHPEND)

Data de cálculo

Se assinalado, o tratamento recalcula o montante do transporte por relação à data de expedi&cced il;ão de cada entidade.

Assim não assinalado, o tratamento re calcula o montante transporte à data especificada.

Data de referência para o re cálculo do montante do transporte.

Por exemplo, se quer calcular o montante do transporte à data da tarifa de n+1.

  • Data de expedição ([field] SHPDAT)
  • Outra data ([field] DAT)

Guia de remessa

Listagens

[By default, the following reports are associated with this function] :

  BLCOUT : Custo de transporte p/GR

  BTCOUT : Custo de transporte p/BT

  RTCOUT : Custo de transporte p/RT

[This can be changed using a different setup].

Barra de menu

  • Pesquisa
  • Cálculo

Pesquisa

Permite lançar a pesquisa dos registos relativos à selecção registada.

Cálculo

Permite lançar o cálculo do custo de transporte das entidades. Sobre cada separador, uma coluna nova que mostra em cartaz o custo calculado à tarifa em vigor.

Após a acção do botão registar, os montantes são registados sobre as entidades apresentadas (BT, RT, BL).

Mensagens de erro

  • [Generic messages]

[In addition to the generic error messages, the following messages can appear during the entry] :

Modo de transporte não associado ao transportador XXX.

O modo de transporte registado não é afectado ao transportador.
