Além das mensagens genéricas, as seguintes mensagens de erro podem aparecer durante o registo :

Erro zona “Estabelecimento”: Ficha inexistente
O valor registado não existe na tabela dos estabelecimentos

Erro zona “Nº campanha RP”: Picking já readjudicado: X
O Nº rearrumação picking registado aqui possui pelo menos um registo já re-adjudicado e não pode por conseguinte ser relançado.

Erro zona “Nº campanha RP”: Rearrumações picking : X, Fiche inexistente
O valor registado não existe na tabela rearrumações pickings para o estabelecimento registado.

Erro zona “Conteúdo”: Recipientes: X, Ficha inexistente
O valor registado não existe na tabela dos recipientes para o estabelecimento registado.

Recipiente: Y X: Nenhum movimento
Para o período do tratamento de cálculo das classes ideais, o sistema não encontrou nenhum movimento histórico e previsional para o recipiente da campanha.

Recipiente: Y X: Campanha rearrumação picking em curso para este recipiente
Existe já uma campanha rearrumação em cursos de tratamento para o recipiente tratado.

Recipiente: Y X: Solicitação sem classe de rotação: Z
A solicitação média por dia calculada para um recipiente artigo da campanha não pôde permitir determinar uma classe de rotação ideal pelo método teórico porque esta solicitação não pertence a nenhum intervalo das solicitações com parâmetros para cada classe de rotação na tabela das classes de rotação.

Conteúdo: Y X: Fluxo volúmico sem classe de volume: Z
O volume médio do picking para um recipiente artigo da campanha não pôde permitir determinar uma classe de volume ideal pelo método teórico porque não compete a nenhum intervalo dos volumes com parâmetros para cada classe de volume na tabela das classes de volume.

Recipiente: Y X: classe de rotação ideal: O mais lugar disponível para o artigo Z
O número de lugares disponíveis é insuficiente em relação ao número de recipientes artigos a re adjudicar e o sistema por conseguinte não pôde atribuir classe de rotação ideal ao recipiente artigo especificado pelo método dos lugares existentes.

Conteúdo: Y X: Classe de volume ideal: O mais lugar disponível para o artigo Z
