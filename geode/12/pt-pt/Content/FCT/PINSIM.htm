Cálculo mnt. de transporte

Esta função permite simular o cálculo do custo de transporte a partir de dados registados.

Esta função pode por conseguinte ser utilizada a título consultivo. Está igualmente na base dos cálculos de custo de transporte, quer pela função de determinação de um transportador, quer no cálculo do custo de transporte para a pré facturação transporte.

Gestão do ecrã

Registo dos diferentes critérios que permitem calcular o custo.

[Tab] Ecrã de registo

Tarifa

Se for conhecido do utilizador, mais simples é registar o N° de tarifa a utilizar para realizar a simulação; se não registar as zonas relativas ao transporte a realizar.

Carregamento

De acordo com o Nº de preço registado de antemão, uma das zonas é apresentada: Trata-se da unidade expressa para definir a tarifa.

Destino

Segundo o Nº de preço registado previamente, as características da zona tarifária são afixadas.

Preço

O preço é apresentado na sequência da acção do botão Cálculo.

Preço

Código tarifa utilizada para calcular o montante do transporte por

funcionalidade "funcion alidade pré-facturação transporte".

Sequência numérica atribuída automaticamente pelo sistema no momento

da tarifa do transportador.

Permite de indicar o número de preços a utilizar para efectuar

cálculo do montan te de transporte.

  • Transportador ([field] CRR)

Código identificando o transportador. Este código deve ser definido na tabela dos transportadores.

  • Modo transpor. ([field] CRRMOD)

Geralmente, o modo de transporte corresponde às noções de servidor de mensagem, fretagem, expresso,... O modo de transporte está verificado por relação à tabela dos modos de transporte.
  • Cód. tarifa ([field] RAC)
  • Número tarifa ([field] CATNUM)
  • Número preço ([field] CPINUM)
