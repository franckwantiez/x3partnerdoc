IMPSOH : Encomendas das vendas

As variáveis são utilizadas em Importação/Exportação dos textos. No quadro da gestão das transações EDI das variáveis são definidas para desencadear a leitura de classes [F] não acessíveis directamente por uma ligação; estas variáveis não são utilizadas que em exportação

Em importação, as variáveis permitem gerar os identificadores EDI para os clientes e o transportador. Nos ficheiros EDI existe um identificador encomenda e um identificador entrega (de mesma estrutura que o identificador encomenda).  Por outro lado em X3 o endereço de entrega é um código endereço ligado ao código do cliente encomenda. Assim em EDI se encontra frequentemente o mesmo identificador para a encomenda e a entrega; fora em X3 deve-se recuperar 2 informações. O software "EDI-Appli" utiliza uma tabela de correspondência e para um identificador EDI entrega, no ficheiro a importar, um único dado reagrupando o código cliente e o código endereço separados por um ~(til).

093195450607700237;AIRFRANCE~RSY (093195450607700237 = identificador EDI  /  AIRFRANCE = código cliente X3  /  RSY = código endereço entrega). Assim o dado AIRFRANCE~RSY está entrregue no ficheiro pivot, unicamente o valor antes do ~ (til) está tomado em conta como código cliente.
