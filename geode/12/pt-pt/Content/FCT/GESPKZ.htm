Zona de preparação:

Este ecrã permite criar, visualizar, modificar, suprimir uma Zona de Preparação de um estabelecimento

A noção de Zona de Preparaçãofoi utilizada para o corte das Ordens de Antecipação.

Pré-requisitos

[Refer to documentation [Implementation]]

Gestão do ecrã

  • Cabeçalho
  • [Tab] Geral
  • [Tab] Preparação

Para criar uma zona de preparação, atribui-se um código que identifica de maneira única sobre 4 caracteres e um intitulado. Depois regista-se as diferentes informações nos 2 separadores presentes.

Cabeçalho

As informações a registar são :

Zona de preparação. um código que o identifica de maneira única sobre 4 caracteres

Intitulado Descrição em claro da zona de preparação.

  • Zona de preparação ([field] PKZ)

Código da zona de preparação que é verificado na tabela das zonas de preparação. A noção de Zona de Preparação é utilizado pelo corte das Ordens de Antecipação.
  • Descr. ([field] PKZDESAXX)
  • Desc. curta ([field] PKZSHOAXX)

[Tab] Geral

Neste separador, encontramos os elementos de edição (Térmica e Laser) e de transmissão versus n1.

Observação sobre a parametrização Impressora: Esta noção permite afectar a cada zona de preparação das impressoras dedicadas e de evitar aos operadores de ter a voltar a um ponto central de edição.

Ao lançamento da preparação:
- Se OP preconizado, imprime as etiquetas pacotes sobre a impressora térmica 1.
- Se OP pré-paletização, segundo o parâmetro "Etiquetas agrupadas" da config. das saidas (Separador Empacotamento):
          - se activo, imprimem-se as etiquetas UM+Pacotes sobre a impressora termica 2.
          - senão, imprimem-se as etiquetas UM sobre impressora térmica 2 e as etiquetas pacotes sobre a impressora térmica 1.

No momento da paletização automática:
- para a OPA (Ordem de Paletização), imprime-se as etiquetas UM sobre a impressora térmica 2.

[Block number] 1

O’conjunto das preparações serão transmitidas ao sistema N1.

  • Transmissão N1 ([field] TRMLVL1)

Ord.

  • Cód. Tri. ([field] CTSCOD)

Edições

  • OP ([field] PKOEDT)

É o código estado que define a edição que será editada em tanto que em Ordem de Preparação. Este código lista verifica na tabela dos códigos Estados.

Impressoras

  • [field] PRN1

Código da impressora versus a qual devem ser encaminhados os documentos de preparação e de aprovisionamento necessitando uma impressora Laser. Este código está verificado na tabela das impressoras.

  • [field] PRN2

Código da impressora versus a qual devem ser encaminhados os documentos de preparação e de aprovisionamento necessitando uma impressora Laser. Este código está verificado na tabela das impressoras.

Ao lançamento da preparação:
- Se OP preconizado, imprime as etiquetas pacotes sobre a impressora térmica 1.
- Se OP pré-paletização, segundo o parâmetro "Etiquetas agrupadas" da config. das saidas (Separador Empacotamento):
          - se activo, imprimem-se as etiquetas UM+Pacotes sobre a impressora termica 2.
          - senão, imprimem-se as etiquetas UM sobre impressora térmica 2 e as etiquetas pacotes sobre a impressora térmica 1.

No momento da paletização automática:
- para OPA (Ordem de paletização), imprime-se as etiquetas UM sobre a impressora térmica 2.
  • [field] PRN3

[Tab] Preparação

Neste separador, encontram-se os critérios da zona de preparação.

Maxima OP

Os valores a seguir são os máximos admitidos para uma ordem de preparação:
Tempo de preparação
Número de linhas
Número de UA
Peso bruto
Volume bruto.

O modo de preparação da OE dá o tipo de OP:

OP picking : todas as antecipações são a realizar a partir do acondicionamento a realizar a partir de acondicionamentos de tipo picking.
OP reserva : todas as antecipações são a realizar a partir de acondicionamentos de tipo reserva.
OP mista : todas as antecipações são a realizar a partir do acondicionamento de tipo picking e reserva :

Em caso de ultrapassagem de um dos critérios fixados, o sistema corta a Ordem de Antecipação de maneira a respeitar os limites estabelecidos.

Cálculo do tempo de preparação. No caso de um movimento reserva, o sistema considera o tempo de destocagem do acondicionamento. No caso de um acondicionamento pickin, o sistema multiplica o tempo de desarmazenamento do acondicionamento por número de objectos antecipados.

OP empacotamento

Os valores a seguir são os máximos admitidos para uma OP, cujas OE o constituam, são com pré-empacotamento:
Grandes pacotes autorizados
Número máximo de pacotes
Peso bruto máximo
Volume bruto máximo
Númeor equivalente de superfície máxima

Em caso de ultrapassagem de um dos critérios fixados, o sistema corta a Ordem de Antecipação de maneira a respeitar os limites estabelecidos.

OP pré-paletização (disponível a contar da versão Premium)

Os valores a seguir são os máximos admitidos para uma OP, cujas OE o constituam, são com pré-empacotamento:
Número máximo de UM
Peso bruto máximo
Volume bruto máximo
Número equivalente de superfície máxima

Em caso de ultrapassagem de um dos critérios fixados, o sistema corta a Ordem de Antecipação de maneira a respeitar os limites estabelecidos.

[Grid] Máx. OP

Máximo admitidos por ordem de preparação "picking". Em caso de ultrapassagem deste critério fixado, o sistema corta a OP de maçneira a res peitar o limite estabelecido

Máximo admitido por uma ordem de preparação em reserva. Em caso de ultrapassagem deste critério fixado, o sistema corta a OP de maçneira a respeitar o limite estabelecido

Máximo admitidos por ordem de preparação mista picking/reserva. Em caso de ultrapassagem deste critério fixado, o sistema corta a OP de maçneira a r espeitar o limite estabelecido

OP Packing

Em modo pré-empacotamento, se os« grandes pacotes«são autorizados na zona de peparação; se não estão autorizados, o pré-empacotamento utilizará os«pequenos pacotes« exclusivamente.

Número máximo de pacotes admitida para uma OP cuja OE o constituindo são gerados em pré-paletização.

Peso bruto no máximo admitido para uma OP cuja OE o constituindo são gerados em pré-empacotamento

Volume bruto máximo admitido para uma OP cuja OE o constituindo são gerados em pré-empacotamento

Equivalente de superfície máximo admitido para uma OP cuja OE o constituindo são gerados em pré-paletização.
