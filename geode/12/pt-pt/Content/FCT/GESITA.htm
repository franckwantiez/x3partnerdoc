ECRA

Gestão do ecrã

A definição das conversões é registada para cada um dos fluxos.

É aconselhado duplicar o existente a fim de conservar parametrização de referência.

As conversões no sentido Warehousing para o nível 3 são prefixadas de X (para exportação).

As conversões no sentido nível 3 para Warehousing não são pré fixadas.

ENTE

Ecrã de registo

Parametrização: Nome do ficheiro como é transmitido ou criado por Warehousing.

Interface

A gerar ou a integrar, a parametrização deve estar activa.

A fila permite ordenar as importações e as exportações para cada um dos fluxos.

É igualmente possível definir um limite horário durante o qual a conversão será activa.

Ambiente

Para cada parametrização de conversão, é possível restringir a lista dos estabelecimentos/recipientes referidos pela conversão.

Assim, pode-se definir se há necessidade dos ficheiros diferentes ligados a modelos diferentes em função dos conjuntos (estabelecimentos, depositantes).

Ficheiros pratos

O nome do modelo associado à conversão deve ser indicado.

O contador está activo apenas para as conversões em exportação.

Por último, é possível definir um caminho diferente do definido em valores parâmetro (capítulo INT) a fim de localizar os ficheiros conversão a tratar e os em rejeição

MS01

Mensagens de erro

Modelo para a conversão XXX.

Um controlo é operado entre o tipo de conversão e o modelo do tipo de conversão registado.
