Paletização vol.

Esta função está disponível a partir G2.

Este ecrã permite paletizar um pacote de maneira simplificada e de retirar de paletes.

A função gestão das U.M é uma função mais completa que permite também gerir paletização e a retirada de palete das embalagens.

A U.M é definida numa tabela parametrização (Tipos de U.M) e comporta um status permitindo a visualização do estado de avanço.

Pré-requisitos

[Refer to documentation [Implementation]]

Gestão do ecrã

O ecrã é constituído de um único separador de registo.

[Tab] Ecrã de registo

Este écran permite:
- de paletizar uma embalagem:
=> sobre uma U.M existente: É necessário então preencher o Nº de U.M, depois o Nº de embalagens a afectar.
=> Sobre uma U.M a criar: É necessário então não informar o Nº de U. M., alterar eventualmente o tipo de U. M seguidamente informar o Nº de embalagens a afectar.
- retirar da palete uma embalagem
Para este facto, é necessário registar o Nº de U.M, assim que o Nº de pacotes

Uma U.M é modificável se ela está aberta.

Se a UM está reportada, o novo pacote afectado deverá ser igualmente reportado.

[Block number] 1

Sequência numérica atribuida automaticamente pelo sistema no momento da validação do’ Cabeçalho OE. Este número é uma informação própria ao es tabelecimento, todos os depositantes simultaneamente. Esta informação constitui uma chave única de acesso a uma Ordem de Entrega.

  • Nr. OE ([field] DLO)
  • Referência ([field] REF)

UM

Número da’Unidade de Manutenção.

Tipo de ’UM sobre o pacote deve ser paletizado.

  • Nr. UM ([field] SHP)
  • Cód. SSCC UM ([field] SHPSSCCOD)
  • Tipo UM ([field] SHU)
  • Coloc. cais ([field] STAQAT0)
  • Armazém ([field] STOQAT0)
  • Endereço ([field] ADDQAT0)

Embalagem

Número do pacote (sequência gerada automaticamente pelo sistema na criação do pacote).

  • Nr. emb. ([field] PRC)
  • Cód. SSCC ([field] SSCCOD)
  • Coloc. cais ([field] STAQAT)
  • Armazém ([field] STOQAT)
  • Endereço ([field] ADDQAT)

Barra de menu

  • Paletizar
  • Despaletizar

Paletizar

Permite validar a acção a paletizar.

Despaletizar

Permite validar a acção de retirar de palete.

Mensagens de erro

  • [Generic messages]

[In addition to the generic error messages, the following messages can appear during the entry] :

Erro zona “Nº de embalagens”: Ficha inexistente

O Nº de pacote registado não existe na tabela das embalagens

Erro zona “Nº UM”: Ficha inexistente

O Nº UM registo não existe na tabela das UM

Erro zona “Nº OE”: Ficha inexistente

O Nº OE registado não existe na tabela das ordens de entrega

Erro zona “Tipo UM”: Ficha inexistente

O tipo UM registado não existe na tabela dos tipos UM

Erro zona “Tipo UM”: Zona obrigatória

O registo do tipo UM é obrigatório.

Embalagem no estado: X - Título do estado

O estado do pacote não permite paletização.

UM reservada ao cliente entregue: XXX e pacotes N° XXX diferentes.

O pacote registado um destinatário diferente do U. M sobre qual foi registado e o U. M deve ser mono destinatário.

Opções de configuração diferentes

As O.E dos pacotes a paletizar não têm a mesma configuração de expedição.

Pacotes não reportados sobre UM reportada

A embalagem registada não é adiada e não pode ser registado sobre uma U.M adiada.

U.M fechada.

Não é possível paletizar ou retirar de palete, paletizar uma embalagem sobre uma U.M já fechada, encarregada ou expedida.
