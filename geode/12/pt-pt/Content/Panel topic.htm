What can I see here?
Symbol

Enter the symbol used to identify the currency, e.g. £ or €.

Name

The name of the currency.

ISO

Select the three-letter ISO code of the currency.

Base Currency

Indicates if this is the base currency in which your company operates.

If you want to change the base currency for your company, click Make Base Currency.

Rate Type

Indicates how the exchange is calculated for this currency.

Options are:

  • Single

    Use a single flat exchange rate.

  • Period

    Use the exchange rate set for a defined period. Enter the different rates and the periods that they apply to.

  • Single & Period

    Use both a flat and period rates for this currency.

Entering exchange rates

The exchange rates entered here are used by default when transactions are entered into Sage 200.

The exchange rates are set to 1 for all currencies by default. You don't have to change them now, but don't forget or this will be the rate used when you start entering transactions.

Add a new currency

If you want to use a different currency, you can add it here now or later. Currencies and exchange rates can be changed at any time.

To add a currency:

  1. Click in the table to make a new line.
  2. Enter the details including the exchange rate if you know it now.
Using period exchange rates

You can also enter exchange rates in date ranges. Any transaction that falls into this date range will use the listed exchange rate.

For the Euro
  1. Select your base currency.
  2. Click in the Period exchange rates to Euro table.
  3. Enter the Expiry date.
  4. Enter the exchange rate.
For other currencies
  1. Select the currency.
  2. Click in the Period exchange rates to £ table.
  3. Enter the Expiry date.
  4. Enter the exchange rate.
Set your base currency

This is the operating currency for your company. This is set the £ by default.

Note: You can't change your base currency once any transactions have been entered.

To set a different currency as your base:

  1. Select the currency.
  2. Click Make Base Currency.
