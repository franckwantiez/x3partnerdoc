Master template - MasterTopNav

Importing files from the master template

The master template is used as a project import to update shared files in other projects; such as stylesheets (CSS), page layouts, and glossaries.

Important

Be very careful when you edit files in the master template project, and make sure that your changes will be suitable for all projects that use the template.

If in doubt, consult the other authors about your change before you make it.

What do you want to do?

Update files in a project that are linked to the master template

To update the master template files in a project:

  1. In Project Organizer, open the Imports folder and the master template import file.

  2. Click Reimport... (on the toolbar) to check for new files.

    Updated files in the master template (files that are newer on the server copy of the master template) are marked Source is newer; these files will be imported.

  3. Click Accept to import the new files.

Import template files in a project (MasterTopNav)

To import the master template in a project:

  1. Select Project > Import File > Add Flare Project Import File.
  2. For the Source, use the default setting New from template, and in the Factory Templates folder select MyProjectImport.
  3. Set the File Name to Master (or anything you want).

  4. Click Add.

    The import file will be added to your project.

  5. Open the import file (in Project Organizer > Imports).
  6. Select the Source Project tab.

  7. Click Browse to select the Source Project, and select:

    \\dell_server\docteam\Flare-templates\MasterTemplateTopNav\MasterTemplateTopNav.flprj
  8. Disable Auto-reimport before "Generate Output". Switching off this setting will allow you to control when updates are applied to your project.
  9. In the Include files section, enter *.* (all files).
  10. In the Import Conditions section, select Edit.

    1. Find the Other - MasterTopNav condition and select Include.

      The import conditions should display include[Other.MasterTopNav].

    2. Enable Auto-Exclude Non-Tagged Files.
  11. To import the files, click Import (at the top).
  12. Click Select All, then click Accept to import the files.
Import optional template files in a project, for setting up a project for first time (MasterTopNavOptional)

Repeat the process above for setting up a project import:

  • All settings should be the same apart from the Import Conditions section.

    1. Select Edit.

    2. Find the Other - MasterTopNavOptional condition and select Include.

      The import conditions should display include[Other.MasterTopNavOptional].

    3. Enable Auto-Exclude Non-Tagged Files.
