Language panel

The following step definitions are available to interact with the main language panel:

the user changes the main language to "(.*)"

the user changes the language to "(.*)" in the more languages panel

the user changes the main language code to "(.*)"

the user changes the language code to "(.*)" in more languages panel

Examples

Interact with the main language panel

Extract from sample: 0311B-LanguagePanelManagement.

#Main language selection by name

Then the user changes the main language to "English (United States)"

 

#Main language selection by code

Then the user changes the main language code to "en-US"

Interact with the more languages panel:

#Language selection by name

Then the user changes the language to "English (United States)" in the more languages panel

 

#Language selection by code

Then the user changes the language code to "en-US" in the more languages panel

The endpoint description could change according to the language. It is therefore important to set the language selection definition before the endpoint selection definition.
