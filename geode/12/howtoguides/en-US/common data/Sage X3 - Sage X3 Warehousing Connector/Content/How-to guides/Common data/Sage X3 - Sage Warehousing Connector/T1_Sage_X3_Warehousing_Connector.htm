Sage X3 - Sage X3 Warehousing connector

Introduction

Communication between Sage X3 and Sage X3 Warehousing is established using an add-on installed on Sage X3. This add-on acts as a connector. An automaton manages data exchanges between the two products using flat files.

This automaton is a data carrier. It monitors the directories and automatically runs tasks to transfer data from Sage X3 to Sage X3 Warehousing, and from Sage X3 Warehousing to Sage X3.

Objectives

This document introduces the logistics flows supported by the connector and the functional limits or constraints when using the connector.

The setup is not covered in this document.

Audience

This document is aimed primarily at functional consultants.
