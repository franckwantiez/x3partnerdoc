Landing pages

New entries must be created manually if the add-on requires adaptations of landing pages.

Creating specific menu items and landing pages

  1. To create multiple types of tiles, use the Menu items function from the Administration navigation page.

    Open: Administration > Authoring > Pages > Menu items

  2. The Link type can be defined as:

    • Query

    • Statistics

    • Process

    • Representation

    • Dashboard page

    • External link

  3. To create new landing pages, use the Home pages function linked to the previously created tiles (menu items):

    Open: Administration > Authoring > Pages > Home pages

Entry

To simplify the management and the extraction of adaptations performed in the Home pages function, a common prefix must be used for the codification of new specific menu items and landing pages of the add-on. The Activity code can be used as a code prefix. You can also manage the translations of these entries if required.

Standard

Sage internal add-ons must be set with the Factory attribute.
