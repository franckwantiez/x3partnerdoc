Delivery and installation

Delivery

The Business Partner License tool is available for download from your local support website. This tool is available on the Sage Knowledge Base website for France and a large part of Europe.

You need to have access to the website.

This tool is versioned and there are no patches. Every change is encapsulated in a new delivery. Ensure that you have the latest version.

Prior to installation

This tool is a web application that will provide encryption mechanisms, enabling you to sign the files provided to you, therefore protecting the files from any unplanned modification. During installation of the tool, an RSA key pair will be generated for the business partner. Different parameters, such as the computer name, will be considered when the key is generated, therefore it is important to select the correct location to install the tool.

It is advisable that you install this tool on a server, shared by team members who will be allowed to create license files. Avoid installing the tool on your laptop.
This tool contains controls in the name of the badges you will create. You will be allowed to create badges with a name that starts with Y, or by your unique identifier. The unique identifier concept is explained in the Developing extensions document.
Prior to installing the Business Partner License tool, ensure that you understand the concepts and have acquired an unique identifier if you need one.

Installation

  1. Click the JAR file and follow the instructions:

  2. Select to install or to re-install. The installer automatically detects an old installation and will propose the re-installation option if applicable:

  3. Select your installation directory:

  4. Select:

    • The Component name (default NODE0). This name should be unique and not already in use.

    • The Service port number (default 8126). This port number should be unique and not already in use.

    • The Administrator account (default admin).

  5. You will then have to select the Passphrase for key pair generation:

  6. Follow the steps to complete the installation:

  7. The installation will create sub-directories in the location selected during installation.

  8. An internet shortcut is available to launch the application:

  9. For a full Syracuse installation, a service will be created and defined as Automatically started.
