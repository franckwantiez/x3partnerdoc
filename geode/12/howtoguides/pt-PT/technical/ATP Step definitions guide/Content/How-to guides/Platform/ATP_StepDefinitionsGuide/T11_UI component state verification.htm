Text field

The following step definitions are available to control the state of a text field:

the selected text field should be enabled

the selected text field should be disabled

the text field with name: "(.*)" is displayed

the text field with name: "(.*)" is hidden

the text field with X3 field name: "(.*)" is displayed

the text field with X3 field name: "(.*)" is hidden
