Exporting and importing customizations

Exporting a customization

To set the filters on the administration data, and to allow a quick extraction of the data associated with navigation pages, landing pages, menu items, and customized pages, access the Customizations management function:

Abrir: Administration > Utilities > Exports > Customizations management

The file created will be in a JSON format, and can be integrated into another Syracuse repository by using the Import Tool.

The translations are also automatically extracted, and are located in the localization section of the JSON file.
The file is stored in the import folder of the server.
