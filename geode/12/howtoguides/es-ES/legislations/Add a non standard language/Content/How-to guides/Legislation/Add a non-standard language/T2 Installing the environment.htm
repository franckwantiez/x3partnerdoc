Installing the environment

The environment to be used to extract and retrieve the translation must be installed as a child folder of the root folder (usually Sage X3).

This folder should contain:

  • The code for at least one standard language.
  • The non-standard language code.
  • The value of the DIS activity code, which must be set to Yes. This activity code activates the translation functions.

The translation folder is the reference folder used to identify the translation differences. It is also used to bring the translation up to date for patches or updates. It is possible at any given moment to know which messages and texts remain to be translated in the software using the untranslated text and message search functions (in the Translation menu).

The validation of each folder generates the standard language dictionary and thus creates the architecture necessary for the import of the non-standard language.
