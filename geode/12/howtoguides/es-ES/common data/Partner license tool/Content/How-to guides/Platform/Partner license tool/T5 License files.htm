Ficheros de licencia

Gestión de licencia

Para poder crear o modificar una licencia, primero debes importar un fichero de partner. Puedes crear un fichero de licencia nuevo o utilizar uno que ya tengas como modelo.

En el caso de que ya haya una licencia, puedes descargar todo el contenido de la creación de la licencia. Se trata de un fichero que tiene el fichero del partner, las reglas y la licencia. Este fichero se procesa como un fichero binario porque su contenido no debe cambiar. Es compatible con el sistema de licencias de Syracuse y se puede utilizar con este. Lo puedes subir desde la función Subida de licencia del módulo Administración de Sage X3.

Una vez que el contenido se ha descargado, ya no puedes modificar la instancia del fichero de licencia porque el sistema considera que este contenido se utilizará en otras partes. No obstante, puedes crear una nueva licencia a partir de este fichero. En la lista de ficheros de licencia también aparece una marca que indica que el contenido de la licencia ya se ha descargado (por tanto, la licencia no se puede modificar).

Cada licencia hace referencia a un conjunto de reglas. Al crear o modificar una licencia, puedes realizar una selección desde la lista de reglas. La estructura de la licencia se adaptará a la estructura de las reglas. Por tanto, actualizar una licencia de cliente con reglas nuevas es sencillo porque los datos vinculados a estas se conservan. Puede haber varias licencia para el mismo cliente y las mismas reglas.

Puedes incluir todas las licencias en la lista o solo las relativas a un cliente en concreto. En este último caso, hay que introducir la referencia del cliente antes de iniciar la búsqueda. Este es el único método con el que puedes ordenar o filtrar los resultados.

Si modificas un fichero de licencia, la nueva versión reemplazará la anterior. El fichero anterior se suprime si el nombre ha cambiado.

Contenido de licencia

El contenido del fichero de licencia se detalla a continuación.

  • Reglas aplicables (creadas por el usuario)

  • Nombre del titular

  • Referencia del titular. Este campo es obligatorio y puede contener letras (A-Z y a-z), números (0-9) y el carácter barra baja (_).

  • Los campos que se muestran a continuación son opcionales para el titular y pueden contener cualquier tipo de carácter:

    • Número de registro

    • Calle

    • Complemento de dirección

    • Ciudad

    • Código postal

    • Región

    • País

  • Si creas una licencia nueva, estos datos, salvo la Referencia, se copiarán desde el fichero del partner. Los siguientes campos son opcionales:

    • Nombre de revendedor

    • Referencia

    • Número de registro

    • Calle

    • Complemento de dirección

    • Ciudad

    • Código postal

    • Región

    • País

  • Para garantizar la validez de la licencia, indica las fechas en los campos Inicio de validez y Fin de validez:

  • Cuando introduces un número en el campo Número de serie, este se compara con el número de serie de la licencia de Sage X3 correspondiente; la licencia solo se acepta si estos números de serie coinciden. De esta forma se evita que la licencia se copie en otro lugar.

  • Selecciona el Tipo de licencia aplicable. Hay varias opciones posibles:

    • Reventa prohibida

    • Versión demo

    • Licencia de partner

  • Para que los Códigos de actividad estén bajo licencia, marca la casilla Licencia. El Código de actividad no se muestra si la casilla Licencia está desmarcada.

  • Introduce las fechas en los campos Inicio de validez y Fin de validez para cada código de actividad que tenga la casilla Licencia marcada. Si no introduces ninguna fecha, la licencia será válida durante todo el periodo de la licencia:
