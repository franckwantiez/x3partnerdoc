Gestión de parches de documentación

Editar elementos para crear un parche

Utiliza la función Creación automática de parches. Esta función permite crear un parche con correcciones de documentación que incluye solo lo que se ha modificado o creado desde una base de datos para un código de actividad determinado.

Para acceder a esta función, ve a Desarrollo > Utilidades > Parches > Creación automática de parches y selecciona solo los códigos ADO, ADZ y ADF.

Consulta la documentación Creación automática de parches (APATCHA).

En la primera pantalla:

  1. Selecciona los códigos ADO, ADF y ADZ.

  2. Introduce las fechas, el código de actividad y el usuario, de ser necesario.

  3. Valida.

En la segunda pantalla (en la columna Número aparece el número de las documentaciones que se van a parchear):

  1. Selecciona la opción Vertical en Tipo de parche.

  2. En el campo Parche, introduce el nombre del fichero de parche.

  3. Si quieres crear el fichero directamente, marca la casilla Generación directa.

Si esta casilla está desmarcada, se crea un fichero que solo incluye la estructura de lo que se va a parchear. Por tanto, es necesario utilizar la función Crear parches para crear la lista de parches (ver más adelante).

Para parchear documentaciones verticales, consulta la documentación de estas funciones:

  • Creación automática de parches (APATCHA)

  • Crear parches (APATCH)

Crear listas de parches

Este procedimiento es el mismo que el utilizado en la Creación automática de parches descrito anteriormente, con la excepción de que la lista de documentación que parchear no se genera automáticamente (a menos que la casilla Generación directa no esté marcada en la función Creación automática de parches). Tienes que indicar, una a una, toda la documentación que quieres integrar en el parche.

Si la casilla Generación directa está marcada, indica el nombre del fichero que incluye la estructura de las documentaciones. Se muestra la lista de documentaciones que se van a parchear; las que no es necesario parchear se pueden retirar.

Esta función está documentada en la ayuda Crear parches (APATCH).

  1. Ve a Desarrollo > Utilidades > Parches > Crear parches.

  2. Selecciona la opción Específico o Vertical en Tipo de parche.

  3. Indica la ruta del parche.

  4. Nombra el parche.

Suprimir la documentación existente en el dossier de destino

La integración de un parche de documentación sobrescribe las documentaciones que ya existen, por lo que no es necesario suprimirlas antes de integrar las nuevas (salvo cuando se trata de documentaciones verticales).

  1. Ve a Desarrollo > Diccionario scripts > Transacciones de sistema.

  2. Para cada transacción: ADF, ADO, ADZ.

Integrar parches

  1. Copia el parche.

  2. Ve a Desarrollo > Utilidades > Parches > Integrar parches.

  3. Selecciona Servidor.

  4. Introduce la ruta para recuperar el parche.

  5. Intégralo en el dossier.

La integración de parches se describe en la documentación sobre la integración de parches en un dossier (PATCH).
