Requisitos previos técnicos generales para instalar los componentes de Sage X3 V12

Java

Los programas de instalación de los componentes de Sage X3 se basan en una estructura de código abierto IzPack. Para poder ejecutarlos, es necesario tener una versión 8 o posterior de Oracle Java Runtime Environment (JRE).

Para poder ejecutar hasta la versión 6.8 de Elasticsearch, se requiere una versión 8 o posterior de Java Development Kit (JDK). La versión 15 de JDK viene integrada a partir de la versión 7.3 de Elasticsearch.

En abril de 2019, la política de Oracle sobre el suministro gratuito de JRE/JDK cambió y ahora es necesario contar con una licencia. Por esto, Sage X3 ha validado las versiones Zulu de JRE/JDK en código abierto (compilaciones OpenJDK). Otras versiones de código abierto podrían funcionar, pero en Sage todavía no se han probado.

.Net Framework

Los siguientes componentes requieren Microsoft .Net Framework 3.5 y 4:

  • Consola de Sage X3
  • Servidor de impresión de Sage X3
