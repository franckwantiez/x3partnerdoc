Text field

The following step definitions are available to write or to verify the value entered in a text field:

the user selects the text field with name "(.*)"

the user selects the text field with X3 field name "(.*)"

the user writes "(.*)" to the selected field

the user writes "(.*)" to the selected text field

the user writes "(.*)" to the selected text field and hits tab key

the user writes "(.*)" to the selected text field and hits enter key

the value of the selected text field is "(.*)"

the value of the selected text field contains "(.*)"

the value of the selected text field is empty

the user clicks the "(.*)" button of the selected text field

the user clicks the "(.*)" button on the "(.*)" text field

the value of the "(.*)" text field is "(.*)"

the user clicks the "(.*)" option of the actions panel

the user writes last day of month to the selected date field and adjusts it by "(.*)"

the user writes first day of the year to the selected date field

the user writes first day of the year to the selected date field and adjusts it by "(.*)"

the user writes last day of the year to the selected date field

the user writes last day of the year to the selected date field and adjusts it by "(.*)"

the user writes last day of "(.*)" to the selected date field

the user writes the "(.*)" from today to the selected date field

the value of the selected date field is month: "(.*)" day: "(.*)" and: year "*(*.)

the value of the selected date field is "(.*)"

the value of the selected date field is today

the value of the selected date field is a generated date using value "(.*)"

the value of the selected date field is the last day of the month

the value of the selected date field is last day of the month adjusted by "(.*)"

the value of the selected date field is the last day of the year

the value of the selected date field is last day of the year adjusted by "(.*)"

the value of the selected date field is the first day of the month

the value of the selected date field is first day of the month adjusted by "(.*)"

the value of the selected date field is the first day of the year

the value of the selected date field is first day of the year adjusted by "(*.)"

the value of the selected date field is the "(*.)" from today

the value of the "(.*)" date field is month: "(.*)" date "(.*)" and year: "(.*)"
