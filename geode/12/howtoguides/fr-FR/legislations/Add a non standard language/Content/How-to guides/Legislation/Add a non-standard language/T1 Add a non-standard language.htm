Add a non-standard language

The Sage X3 technology software is used to manage a group of standard languages, which are supplied with the system. However, it is possible to add additional languages manually.

This procedure does not guarantee the evolution of a language's messages on delivery of a standard patch list, which is likely to add new messages. In this case, it is necessary to add these additional messages to the non-standard language which has been created.

It is strongly recommended to export the texts in the new language from the APLSTD and ATEXTE tables to a backup file once the process has been carried out, in order to be able to import them again if necessary.

Les images écran peuvent être différentes en fonction de la version du produit et/ou de la configuration utilisées.
