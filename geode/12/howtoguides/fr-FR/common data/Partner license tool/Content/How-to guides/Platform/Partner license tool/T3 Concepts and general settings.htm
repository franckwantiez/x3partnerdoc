Concepts et paramétrages généraux

Vue d'ensemble

L'Outil de licence partenaire est une application web livrée dans les paramétrages. Il vous permet de créer, de modifier et d'afficher des fichiers de règles et de licence.

Un fichier de règles décrit une application, son contenu, et les éléments qui doivent être contrôlés. Il est lié à un produit et peut être versionné.

Un fichier de licence est accordé à un client. Il s'appuie sur un fichier de règles, et stocke les informations que vous avez décidé de contrôler dans les règles, par exemple, le nombre de badges autorisés.

Vous pouvez télécharger n’importe quel fichier de licence et il contiendra le fichier de règles ainsi que le fichier de votre partenaire. Tous les fichiers sont automatiquement signés de façon électronique grâce à votre clé privée.

La gestion des règles et des licences est verrouillée jusqu'à ce que vous téléchargiez le fichier partenaire, signé électroniquement par Sage X3. La procédure de récupération du fichier est expliquée dans le chapitre précédent, Livraison et installation.

Fichier partenaire

Lors de l'installation de l'outil, une clé RSA est générée pour le partenaire. La clé publique doit être envoyée à Sage X3, qui va générer et apposer une signature électronique sur un fichier partenaire, puis renvoyer ce fichier au partenaire concerné.

Il n'est pas possible de créer ou d'éditer des fichiers de règles ou de licence tant que l'outil n'a pas récupéré le fichier partenaire.

Le fichier partenaire peut être téléchargé dans la section Général. Ce fichier a des répercussions sur les règles de nommage des badges. En effet, un nom de badge peut commencer soit par Y, soit par l'un des préfixes de la propriété dictionaryPrefix. Par exemple, lorsque la valeur de dictionaryPrefix est XNB-XND,XX01-XX02,XZ, le nom du badge peut être : XNB*, XNC*, XND*, XX01*, XX02*, XZ*, ou Y*. L'astérisque (*) représente un nombre arbitraire de caractères compris entre A-Z et 0-9.

Gestion des utilisateurs

Pour utiliser l'outil, vous devez d'abord vous authentifier dans le navigateur. Renseignez les informations suivantes :

  • Identifiant

  • Nom

  • Mot de passe

  • Indicateur 'Utilisateur administrateur' sur Oui/Non

Une fois la connexion effectuée, le nom de l'utilisateur s'affiche dans l'application. Les utilisateurs administrateurs peuvent créer et supprimer d'autres utilisateurs et définir des mots de passe. Les utilisateurs non-administrateurs peuvent uniquement afficher et modifier leurs propres données (changer leur mot de passe, par exemple). Ce sont les seules différences entre ces deux types d'utilisateurs.

Si votre navigateur n'est pas fermé, vous restez connecté.

Paramètres régionaux

Un fichier de règles inclut un certain nombre d'éléments régionaux différents. Pour une saisie plus rapide des règles, vous pouvez indiquer des paramètres régionaux standard qui s'afficheront systématiquement.

Informations techniques

Cet outil utilise JavaScript dans le navigateur, mais pas les cookies. Pour vous déconnecter, fermez le navigateur complet, et pas seulement un onglet, ou le navigateur continuera de stocker vos informations d'authentification.

Cet outil exploite les fichiers stockés dans le répertoire et les sous-répertoires de l'outil de licence. Il n'utilise pas de base de données.
