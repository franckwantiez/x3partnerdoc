Annexe : Exemple d'import de classeur

Importer un fournisseur

Cet exemple illustre comment importer les données fournisseur d'un classeur dans Sage X3.

Exporter un modèle d'import Sage X3 dans un logiciel tableur

Ouvrir : Paramétrage > Exploitation > Imports/Exports > Modèles import/export | Champs

  1. Ouvrez le modèle BPS - Fournisseurs.
  2. Cliquez sur Export Excel depuis l'icône Actions pour exporter les champs associés.
  3. Cliquez sur Exporter.

    4. Une icône correspondant à votre logiciel tableur s'affiche. Dans cet exemple, il s'agit de Microsoft Excel.

  4. Cliquez sur l'icône Microsoft Excel.

Préparer un nouveau classeur

Ouvrir : Microsoft Excel

  1. Ouvrez votre classeur.


  2. Sélectionnez les cellules à copier.

    3. Dans notre exemple, nous souhaitons uniquement traiter les enregistrements des quatre premières colonnes : indicateur, table, champ et intitulé (Indicator, Table, Field et Description).

  3. Copiez les cellules sélectionnées dans votre presse-papier.
  4. Ouvrez un nouveau classeur vide. Sélectionnez Collage spécial. Sélectionnez la case à cocher Transposé.

    5. Laissez les options Coller > Tout et Opération > Aucune activées.

  5. Vous remarquez alors que les données des colonnes copiées sont présentées de façon horizontale.

  6. Surlignez les cellules dont la première ligne contient (ligne Indicator) contient la lettre B. Appliquez une Couleur de remplissage.
  7. Répétez l'étape 6 pour toutes les cellules dont la première ligne (ligne Indicator) contient la lettre A.
  8. Surlignez la colonne A et cliquez sur Insérer pour ajouter une nouvelle colonne.
  9. Dans la cellule A1, saisissez H.

    10. Rappel : Ligne d'en-têteH (ligne d'en-tête) est obligatoire dans la première cellule (première ligne de la première colonne).

  10. Dans la cellule A2 et toutes les lignes suivantes de la colonne A, saisissez C.

    11. Rappel : Les lignes commentairesLes lignes C (commentaires) sont ignorées par la fonction Conversion fichiers mono-ligne (AIFIMPGENFIC).

  11. Appliquez le formatage de la colonne B aux cellules de la colonne A.

  12. Supprimez la colonne B.
  13. Conservez la valeur B dans la nouvelle cellule B1. Supprimez la lettre B de toutes les autres cellules de la première ligne.

  14. Conservez la première occurrence de la lettre A sur la première ligne. Supprimez la lettre A de toutes les autres cellules de la première ligne.

  15. Appliquez les mêmes étapes pour la lettre R de la première ligne.
  16. Copiez toutes les cellules de toutes les lignes de la section A de une à N fois, en insérant chaque bloc copié avant la première colonne de la section R.
  17. Copiez toutes les cellules de toutes les lignes de la section R de une à N fois.
    18. Il n’y a aucune limite sur le nombre de sections copiées (sections A ou R).
  18. Dans la dernière cellule de la première ligne, saisissez FIN.

    19. Il s’agira de la dernière cellule de la première ligne de la dernière section R.

    Rappel : FINLa dernière colonne doit contenir la valeur FIN.

    Sage vous recommande d’appliquer une couleur de remplissage à cette dernière cellule (FIN).

Charger les données dans le nouveau classeur

Ouvrir : Microsoft Excel

Ouvrir : Exploitation > Imports/Exports > Exports

  1. Dans la fonction Exports (GEXPOBJ), sélectionnez le modèle BPS - Fournisseurs comme Modèle à utiliser.
  2. Bloc Bornes, champ Fournisseur. Indiquez le même code fournisseur dans les champs Début et Fin.
    3. Vous ne devez exporter qu’un seul fournisseur.
  3. Ouvrez Microsoft Excel et utilisez l’Assistant importation de texte pour ouvrir votre fichier d’export.
  4. Copiez les lignes de type B, A et R, selon vos besoins.

    5. Notre exemple ne concerne qu’un seul fournisseur.
  5. Renseignez les lignes pour chacun des fournisseurs concernés.
    6. H B      
    C BPSUPPLIER BPSUPPLIER BPSUPPLIER BPARTNER
    C / BSGCOD BPSNUM BPRNAM
    C   Catégorie Fournisseur Raison sociale
      B NEG FR001 FOURNISSEUR DE NEGOCE
      B NEG TST001
