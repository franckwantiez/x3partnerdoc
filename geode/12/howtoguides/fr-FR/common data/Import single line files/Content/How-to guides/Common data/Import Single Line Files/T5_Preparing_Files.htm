Préparer un fichier mono-ligne pour l’import

Définition d'un fichier CSV

Un fichier CSV est un fichier dans lequel les valeurs sont délimitées par des virgules. Les noms des fichiers CSV se terminent par l’extension de fichier .csv.

Ce sont des fichiers de texte brut, qui enregistrent les données sous forme de tableau. Ils se présentent, en général, sous la forme de feuilles de calcul compatibles avec des logiciels comme Microsoft Excel ou Google Sheets. Ces fichiers permettent notamment d’exporter ou d'importer un grand volume de données dans une base de données.

Un classeur CSV contient une unique feuille. Il n’est pas possible d’enregistrer des cellules, colonnes, lignes ou formules, dans un fichier CSV.

Processus

Pour convertir un fichier mono-ligne à importer dans Sage X3, vous devez suivre les deux étapes suivantes :

Étape 1 : Enregistrer le fichier source au format CSV

Dans cet exemple, le fichier source est un classeur nommé exemple.xlsx.
  1. Ouvrez le fichier de données source à convertir dans l'application associée.

  2. Depuis l’application, cliquez sur Fichier > Enregistrer sous.
  3. Sélectionnez le format de fichier CSV (délimité par des virgules) (*.csv).

  4. Cliquez sur Enregistrer.

Étape 2 : Convertir le fichier source CSV en fichier CSV multi-ligne

Ouvrir : Exploitation > Imports/Exports > Conversion fichiers mono-ligne

Saisissez les informations suivantes :

  • Bloc Structure :
    • Les champs de ce bloc indiquent que le fichier à lire est au format CSV.

    Séparateur de champ : Conservez le point-virgule (;).

    Sép d'enregistrement : Valeurs par défaut associées à un système Windows.

    Format fichier : Conservez ascii.

    Pour cet exemple, il n’est pas nécessaire de modifier les valeurs de ce bloc.
  • Bloc Source :
    • Ce bloc vous permet de charger le fichier source.

    Client (par défaut). Conservez Client si le fichier source est localisé sur le poste du client.

    1. Cliquez sur OK.
    2. Sélectionnez le fichier .csv source à importer. Vous pouvez également glisser-déposer le fichier dans la fenêtre Téléchargement de fichiers.
      3. Assurez-vous de sélectionner le fichier avec l’extension .csv et non le fichier de données d’origine.
    3. Une fois le fichier chargé, cliquez sur OK.
    4. Lorsque l'import est effectué, un fichier de trace s'affiche indiquant le nom du fichier importé.

      5. Dans cet exemple, le nom du fichier d'import est : exemple.csv.out.

    Serveur : Sélectionnez le bouton radio Serveur si le fichier source se trouve sur le serveur Sage X3.

    1. Fichier de données : Saisissez ou sélectionnez le fichier .csv source à importer dans la Liste des volumes.
      2. Assurez-vous de sélectionner le fichier avec l’extension .csv et non le fichier de données d’origine.
    2. Cliquez sur OK.
    3. Le champ Fichier de données du bloc Destination affiche le nom de fichier automatiquement. Il n’est pas modifiable.

      4. L’emplacement du fichier cible sera le même que celui du fichier source. Le nom du nouveau fichier converti contiendra l’extension .out.

      Dans cet exemple, le nom du fichier d'import est : exemple.csv.out. Il sera sauvegardé dans le même dossier que le fichier source sur le serveur Sage X3.

    4. Cliquez sur OK.

Allez à l’emplacement du fichier .csv.out. Si le fichier a été généré sur le poste du client, il se trouvera dans le dossier Téléchargements ; s’il a été généré sur le serveur Sage X3, il se trouvera dans le même dossier que le fichier source.

Le format du fichier .csv.out est compatible avec le format (CSV) de l'application. C’est ce fichier que vous utiliserez pour importer les données dans l'application.
