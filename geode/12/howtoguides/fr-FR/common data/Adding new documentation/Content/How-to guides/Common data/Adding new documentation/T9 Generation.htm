Génération sur le serveur de documentation

Si les nouvelles documentations concernent un menu, une table, un menu local ou un type de donnée, une mise à jour du type correspondant (APM, ATB, AML, ATY) doit être effectuée.

  1. Dans la fonction Documentation (GESADO), cliquez sur Génération.

  2. Renseignez le type de documentation.

  3. Dans le bloc Génération ADOCUMENT, sélectionnez la case à cocher Toutes.

  4. Dans le bloc Génération document final, sélectionnez Complète ou Selon code activité, et Aides sur champs liées aux documentations.

Pour pouvoir accéder à la documentation depuis le produit, vérifiez le format d’accès à la documentation. Si le format compressé est utilisé, il faut re-compresser la documentation via la console de configuration (consultez l'aide sur le Serveur de documentation (SRVDOC)).

Le format d’accès à la documentation est indiqué dans la boîte de connexion au dossier (cliquez sur l'action Avancé pour ouvrir la fenêtre correspondante).

  • Si le format est Format CHM, la documentation est lue dans son format compressé (lecture du fichier CHM).

  • Si le format est Format HTM la documentation est consultable immédiatement.

Après l'installation du patch, regénérez les répertoires HTM.

Après la regénération, suivez cette procédure  :

  1. Ouvrez la console de configuration.

  2. Vérifiez le serveur de documentation.

  3. Cliquez sur la section Configuration.

  4. Cliquez sur le bouton Compresser (CHM).
