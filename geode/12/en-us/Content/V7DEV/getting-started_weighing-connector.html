    • Message – All scales need to be calibrated


    Go to Weigh station, change the Calibration code to No Calibration

    The other options will not work if we don’t have an actual scale connected






    Message – Allocation invalid lot mixing prohibited


    Solution: Create another misc receipt for the component by choosing a different lot.Then manually allocate the component with this new lot.

    Message – Cannot modify Work order being weighed



    Message – Cannot modify Work order being weighed.

    Go to Weighing situation, enter your FGSelect the Action dropdown and Reactive the WO.

    Click Yes on message


    Message – Component already fully weighed when it is not

    "Component already fully weighed":



    When you get this message, verify component unit of measure has weight unit and item weight entered

    You didn’t have a weight unit and a weight setup on the Product for the component.I entered the information AND had to create a new WO




    When weight see the Allocated quantity and Quantity to weigh are “0”

    Message – Conv. Not found


    Need to create a UOM conversion


    Fix error message - maximum number of users of this function has been reached


    Development > Utilities > Verifications > System monitor > Users

    Find the Open session FUNSCA and select it


    On the Active processes, select the process no. and click the Action button and select Process Termination (Or closing session)


    If you are getting this messages – No extracted operation or WO being weighed



    Verify that your Operation is assigned to the component in BOM.

    Have to create a new work order after you fix the BOM.
