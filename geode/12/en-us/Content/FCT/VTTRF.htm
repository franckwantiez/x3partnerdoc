Transfer

This function is used to validate all or part of a transfer/reorder movement according to the transfer process in 2 phases (Picking, Relocation).

Tab Transfer no.

The operator must specify the phase (Picking or Relocation), the Transfer no. or the Movement no..
In the first case, the system will list those movements associated with the transfer no. and pending phase selection, with the possibility being provided for each to process it via theor to move to the next via the.

Phase
- Mandatory field
- Specify the phase to be carried out: Picking or Relocation

Transfer no
- Optional field
- Specify a non-validated transfer no. (for which there is at least one movement with status < 4 and pending phase selection)
submits the corresponding transfers.

Movement no.
- Optional field that can only be entered if the Transfer no. has not been entered
- Specify the number of a non-validated transfer/reorder movement (status < 4) and pending phase selection.
submits corresponding movements.

Tab Deposit (Relocation)

Item
- item of the movement
- this field can be assigned according to the Site setup (Item code and/or EAN code checked). In that case, enter the item code or its EAN code.

Number/Cnt and Number/Homogeneous Cnt
- nb/container of the displayed movement
- nb/homogeneous container, which can be entered except for a full-container type transfer
- the homogeneous nb must be positive, not equal to zero and at the most equal to the nb of PAC units of the movement
- the homogeneous cnt must be mandatory; It must belong to the palletization program of the Cnt
- The validated PAC unit nb (= homogeneous nb * standard nb of PAC units of the homogeneous cnt) must be :
  . at the most equal to the nb of PAC units of the initial movemen
  . multiple of the homogeneous quantity of the Cnt
  . multiple of the PAC unit nb of the reference cnt of the picking/relocation location if one or the other is the picking 
- if the picking store comes with full object transfer, partial transfer is only possible if the cnt coefficient (=nb PAC/nb cnt) remains unchanged with respect to the initial transfer.
submits the list of containers of the Cnt palletization program.

BC/Code for Picking location
- mandatory field
- enter or scan the location or its control code depending on the setup.

Position no.
- Optional field
- enter the physical position of the picking on the handling equipment used by the operator.

moves the transfer/reorder movement to status 7 (Under progress) by previously splitting it in 2 in the event of a partial validation. The quantities of picking locations and stock objects are also modified (see the 2-phase transfer process), along with the status of the merging-of-flow type wave that may be associated.
Depending on the setup of the site and stock configuration, this operation can require that an operator code be entered.

Tab Relocation

Position no.
- this field cannot be entered: the system suggests the movements to be relocated acccording to the increasing order of the position no.

BC/Code for Relocation location
- mandatory field
- enter or scan the location or its control code depending on the setup.

Item
- item of the movement
- this field can be assigned according to the Site setup (Item code and/or EAN code checked). In that case, enter the item code or its EAN code.

Number/Cnt and Number/Homogeneous cnt
- this field cannot be entered. Only the exact quantity picked can be validated on relocation.

moves the transfer/reorder movement to status 8 (Validated). The quantities of the relocation locations and stock objects and of the temporary location are also modified (see the 2-phase transfer process), along with the status of the merging-of-flow type wave that may be associated.
Depending on the setup of the site and stock configuration, this operation can require that an operator code be entered.

Specific Buttons

Next

This button is only activated if a transfer no. has been entered in the first tab. It makes it possible to go to the movement following the latter, without any quantity being validated on the current movement.
