Tariff codes

This function is used to manage (create, modifiy or delete) tariff codes for the current site.

Tariff code definition: a tariff code is used to assign a type of tariff to a given section.

This function is part of the optional module GEOPLO for the invoicing of logistic services.

Prerequisite

Refer to documentation Implementation

Screen management

The screen is constituted of a table displayed in a single tab. The tariff codes are then entered in the table.

    Remarks on the management rules of tariff codes:
  • Modification Mode: a tariff code can only be modified if all associated invoices are validated.
  • Deletion Mode: A tariff code can only be deleted if it is not linked to any logistic service.

Entry screen

 

Field descriptions

Grid

Price list code (field IPCCOD)

Price-list code used to calculate the field amount in the GEOPLO functionality (logistic service pre-invoicing module).

Description (field IPCDESAXX)
Heading (field IHHCOD)

Section code defined in GEODE section table.

This section is a service invoicing element such as "the number of DN shipped", "the number of parcel shipped", "the weight of pallets received", etc.
Unit (field IPCUOM)

Indicates the tariff code unit.

Fyi: the units are managed in the Common table for Units of Measure.
