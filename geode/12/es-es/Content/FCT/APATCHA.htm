Creación automática de parches

Esta función permite crear un fichero de parche que integra todos los elementos creados o modificados entre dos fechas, si es necesario marcados mediante un código de actividad.

Se puede definir el tipo de elementos que se desea parchear y proceder de dos maneras diferentes:

  • mediante una generación directa del fichero (listándose el número de elementos presentes por tipo),
  • o mediante la generación de un fichero de parche que sólo contiene la lista de elementos. En este caso, se releerá este fichero en la función de parche para cargar la lista completa de los elementos afectados y permitir su modificación si es necesario.

Ventana de selección preliminar

Al acceder a la función aparece una primera ventana que permite definir los criterios de búsqueda de los elementos a parchear.

Criterios

Indicar la fecha de inicio para seleccionar los objetos a parchear.

Esta se comparará con la fecha de modificación (o con la fecha de creación para los nuevos objetos).

Sí a todo lo demás

Cuando hay una tabla de introducción de múltiples opciones con una opción de respuesta Sí / No en cada línea, esta función permite responder Sí para todas las líneas a partir de la línea actual.

Ventana de validación

Una vez introducida y validada la primera ventana, se ejecuta el proceso de selección (una ventana de progresión muestra la selección en curso) y, al cabo de unos instantes, se abre una segunda ventana. En esta ventana aparecen las informaciones relativas al resultado de la búsqueda y se puede completar la introducción de datos para crear directamente el fichero de parche.

Tipo de parche

  • Tipo de parche (campo TYPPTC)

Fichero

  • campo AW
  • Tipo de destino (campo TYPEXP)
  • Parche (campo VOLFIL)
  • Generación directa (campo FLAGPAT)

Bloque Número 3

  • Fecha fin (campo DATFIN)
