Número max. conexiones

Esta parámetro permite bloquear una cuenta de usuario más allá de un cierto número de tentativas de conexión con una contraseña incorrecta. Puesto que no es nulo para el usuario, si se intenta la conexión un número de tentativas infructuosas que sobrepasan el valor autorizado, se bloqueará la cuenta del usuario. Para desbloquearla, el administrador tiene dos soluciones posibles:

  • pasar por la función de supresión de contraseña en gestió n de los usuarios, que desbloqueará la cuenta. Por motivos de seguridad, se recomienda imponer el uso de contraseña para que el usuario, en su próxima conexión, introduzca una nueva contraseña (que puede definirse por el parámetro PASSWD).
  • aumentar el valor de este parámetro (al menos temporalmente) para el usuario, para permitirle disponer de intentos suplementarios. Por ejemplo, pasando el valor de 3 a 5, y el usuario dispondrá de dos oportunidades más para conectarse. Cuando se establezca la conexión, el número de intentos se pone en cero.

Nivel de localización / Variable global

Este parámetro está definido a nivel [Folder]. Pertenece al capítulo SUP (Supervisor) y al grupo SEC (Seguridad), Los parámetros siguientes se encuentran también asociados a este capítulo y grupo :

  • ADMUSR (Superusuario)
  • CHGPASS (Cambio contraseña)
  • MAILSAGE (Dirección mail licencia Sage)
  • PASLNG (Longitud contraseña)
  • PASSWD (Contraseña obligatoria)
  • SSOCLAOBJ (Clase para búsqueda LDAP)
  • SSOCONNECT (Conexión SSO)
  • SSODOMAIN (Dominio usuario)
  • SSOMAJ (ACT usuario en la conexión)
  • SSOPASSWD (Control contraseña SSO)
  • SSOREFMAJ (Referencia para Act. usuario)
  • TABTRA (Sgmto.operaciones)

Parámetros enlazados

Los parámetros siguientes están vinculados al parámetro NBRCON  :

Supervisor

  PASLNG (definido a nivel [Folder]) : Longitud contraseña

  PASSWD (definido a nivel [Folder]) : Contraseña obligatoria

  TIMEHGUP1 (definido a nivel [User]) : Tiempo límite para desconexión

  TIMEHGUP2 (definido a nivel [User]) : Tiempo de espera desconexión

  TIMEHGUP3 (definido a nivel [User]) : Time-out sesión segundario

Funciones afectadas

Afecta a la dase de conexión de un usuario.

Notas

Este tipo de control se aplica a todos los usuarios independientemente de su perfil (incluidos administradores definidos por el parámetro ADMUSR), con excepción del usuario administrador general.
